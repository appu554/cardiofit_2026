# 🚀 **Module 5: ML Inference & Real-Time Risk Scoring - Complete Design**


## 📋 **Executive Summary**


**Goal:** Integrate machine learning models for real-time risk prediction, embedding trained models directly into the Flink streaming pipeline for sub-second inference.


**Timeline:** 3-4 weeks (120-160 hours)  
**Complexity:** High (ML + Streaming + Feature Engineering)  
**Prerequisites:** Module 4 Complete (CEP + Windowed Analytics operational)


---


## 🎯 **Module 5 Objectives**


### **What We're Building:**


1. **ML Model Integration** (1 week)
   - ONNX Runtime embedding in Flink
   - Model versioning and hot-swapping
   - Batch inference optimization
   - Model performance monitoring


2. **Feature Engineering Pipeline** (1 week)
   - Real-time feature extraction
   - Feature store integration
   - Temporal feature windows
   - Feature drift detection


3. **Risk Prediction Models** (1 week)
   - Mortality prediction (APACHE IV-based)
   - Readmission risk (HOSPITAL+ score)
   - Sepsis onset prediction (InSight model)
   - AKI progression (KDIGO-ML)
   - Clinical deterioration (LSTM-based)


4. **Model Serving Infrastructure** (1 week)
   - A/B testing framework
   - Shadow mode deployment
   - Model explainability (SHAP values)
   - Feedback loop for retraining

### **5A.1: Model Container**


```java
package com.hospital.flink.ml;


import ai.onnxruntime.*;
import lombok.Data;
import java.nio.FloatBuffer;
import java.util.*;


/**
 * ONNX Model wrapper for Flink streaming
 * Handles model loading, inference, and lifecycle
 */
@Data
public class ONNXModelContainer implements Serializable {
    
    private final String modelId;
    private final String modelName;
    private final ModelType modelType;
    private final String modelVersion;
    
    // ONNX Runtime components
    private transient OrtEnvironment environment;
    private transient OrtSession session;
    private transient Map<String, OnnxTensor> inputCache;
    
    // Model metadata
    private final List<String> inputFeatureNames;
    private final List<String> outputNames;
    private final ModelConfig config;
    
    // Performance tracking
    private transient long inferenceCount = 0;
    private transient long totalInferenceTime = 0;
    
    public enum ModelType {
        MORTALITY_PREDICTION,
        SEPSIS_ONSET,
        READMISSION_RISK,
        AKI_PROGRESSION,
        CLINICAL_DETERIORATION
    }
    
    /**
     * Initialize ONNX Runtime session
     * Called once per TaskManager (not per record)
     */
    public void initialize() throws OrtException {
        this.environment = OrtEnvironment.getEnvironment();
        
        // Load model from resource or external storage
        byte[] modelBytes = loadModelBytes();
        
        // Create session with optimization
        OrtSession.SessionOptions options = new OrtSession.SessionOptions();
        options.setOptimizationLevel(OrtSession.SessionOptions.OptLevel.ALL_OPT);
        options.setIntraOpNumThreads(4);  // Parallel inference
        options.setInterOpNumThreads(2);
        
        this.session = environment.createSession(modelBytes, options);
        this.inputCache = new HashMap<>();
        
        System.out.println(String.format(
            "Initialized ONNX model: %s v%s (inputs: %d, outputs: %d)",
            modelName, modelVersion, 
            session.getInputNames().size(),
            session.getOutputNames().size()
        ));
    }
    
    /**
     * Perform inference on a single patient
     * 
     * @param features Feature vector (must match model input schema)
     * @return Prediction output (probability, risk score, etc.)
     */
    public MLPrediction predict(float[] features) throws OrtException {
        long startTime = System.currentTimeMillis();
        
        // Validate input dimensions
        if (features.length != inputFeatureNames.size()) {
            throw new IllegalArgumentException(String.format(
                "Feature dimension mismatch: expected %d, got %d",
                inputFeatureNames.size(), features.length
            ));
        }
        
        // Create input tensor
        long[] shape = {1, features.length};  // Batch size = 1
        FloatBuffer buffer = FloatBuffer.wrap(features);
        OnnxTensor inputTensor = OnnxTensor.createTensor(environment, buffer, shape);
        
        // Run inference
        Map<String, OnnxTensor> inputs = Map.of(
            session.getInputNames().iterator().next(),
            inputTensor
        );
        
        OrtSession.Result result = session.run(inputs);
        
        // Extract output
        float[] outputArray = extractOutput(result);
        
        // Build prediction object
        MLPrediction prediction = MLPrediction.builder()
            .modelId(modelId)
            .modelType(modelType)
            .modelVersion(modelVersion)
            .prediction(outputArray[0])  // Main prediction (e.g., mortality probability)
            .confidence(calculateConfidence(outputArray))
            .features(features)
            .featureNames(inputFeatureNames)
            .inferenceTimeMs(System.currentTimeMillis() - startTime)
            .timestamp(System.currentTimeMillis())
            .build();
        
        // Update metrics
        inferenceCount++;
        totalInferenceTime += prediction.getInferenceTimeMs();
        
        // Cleanup
        inputTensor.close();
        result.close();
        
        return prediction;
    }
    
    /**
     * Batch inference for multiple patients
     * More efficient than individual calls
     */
    public List<MLPrediction> predictBatch(List<float[]> featureBatch) throws OrtException {
        long startTime = System.currentTimeMillis();
        
        int batchSize = featureBatch.size();
        int featureCount = inputFeatureNames.size();
        
        // Create batched input tensor
        float[] flattenedFeatures = new float[batchSize * featureCount];
        for (int i = 0; i < batchSize; i++) {
            System.arraycopy(featureBatch.get(i), 0, 
                           flattenedFeatures, i * featureCount, 
                           featureCount);
        }
        
        long[] shape = {batchSize, featureCount};
        FloatBuffer buffer = FloatBuffer.wrap(flattenedFeatures);
        OnnxTensor inputTensor = OnnxTensor.createTensor(environment, buffer, shape);
        
        // Run batch inference
        Map<String, OnnxTensor> inputs = Map.of(
            session.getInputNames().iterator().next(),
            inputTensor
        );
        
        OrtSession.Result result = session.run(inputs);
        float[][] outputs = extractBatchOutput(result, batchSize);
        
        // Build prediction objects
        List<MLPrediction> predictions = new ArrayList<>();
        long batchInferenceTime = System.currentTimeMillis() - startTime;
        
        for (int i = 0; i < batchSize; i++) {
            predictions.add(MLPrediction.builder()
                .modelId(modelId)
                .modelType(modelType)
                .prediction(outputs[i][0])
                .confidence(calculateConfidence(outputs[i]))
                .features(featureBatch.get(i))
                .inferenceTimeMs(batchInferenceTime / batchSize)  // Amortized
                .timestamp(System.currentTimeMillis())
                .build());
        }
        
        // Cleanup
        inputTensor.close();
        result.close();
        
        return predictions;
    }
    
    /**
     * Extract single prediction output from ONNX result
     */
    private float[] extractOutput(OrtSession.Result result) throws OrtException {
        OnnxValue outputValue = result.get(0);
        float[][] output2D = (float[][]) outputValue.getValue();
        return output2D[0];  // First (and only) batch element
    }
    
    /**
     * Extract batch predictions from ONNX result
     */
    private float[][] extractBatchOutput(OrtSession.Result result, int batchSize) 
        throws OrtException {
        OnnxValue outputValue = result.get(0);
        return (float[][]) outputValue.getValue();
    }
    
    /**
     * Calculate confidence score from model output
     * For binary classification: max(p, 1-p)
     * For multi-class: max(softmax probabilities)
     */
    private double calculateConfidence(float[] output) {
        if (output.length == 1) {
            // Binary classification
            float probability = output[0];
            return Math.max(probability, 1 - probability);
        } else {
            // Multi-class classification
            float max = Float.MIN_VALUE;
            for (float prob : output) {
                max = Math.max(max, prob);
            }
            return max;
        }
    }
    
    /**
     * Get model performance metrics
     */
    public ModelMetrics getMetrics() {
        return ModelMetrics.builder()
            .modelId(modelId)
            .inferenceCount(inferenceCount)
            .averageInferenceTimeMs(
                inferenceCount > 0 ? (double) totalInferenceTime / inferenceCount : 0
            )
            .throughputPerSecond(
                totalInferenceTime > 0 ? (inferenceCount * 1000.0) / totalInferenceTime : 0
            )
            .build();
    }
    
    /**
     * Load model bytes from storage
     * In production: Load from S3, GCS, or model registry
     */
    private byte[] loadModelBytes() throws OrtException {
        try {
            // Option 1: Load from classpath (embedded in JAR)
            InputStream modelStream = getClass().getResourceAsStream(
                "/models/" + modelId + ".onnx"
            );
            
            // Option 2: Load from external storage (production)
            // modelStream = s3Client.getObject(bucketName, "models/" + modelId + ".onnx");
            
            return modelStream.readAllBytes();
            
        } catch (IOException e) {
            throw new OrtException("Failed to load model: " + modelId, e);
        }
    }
    
    /**
     * Cleanup ONNX resources
     * Called when TaskManager shuts down
     */
    public void close() throws OrtException {
        if (session != null) {
            session.close();
        }
        if (environment != null) {
            environment.close();
        }
        
        System.out.println(String.format(
            "Closed ONNX model: %s (processed %d inferences, avg latency: %.2f ms)",
            modelName, inferenceCount,
            inferenceCount > 0 ? (double) totalInferenceTime / inferenceCount : 0
        ));
    }
}


/**
 * ML Prediction result
 */
@Data
@Builder
public class MLPrediction {
    private String modelId;
    private ONNXModelContainer.ModelType modelType;
    private String modelVersion;
    
    // Primary prediction
    private double prediction;           // Main output (e.g., mortality probability 0.0-1.0)
    private double confidence;           // Model confidence in prediction
    
    // Input features (for explainability)
    private float[] features;
    private List<String> featureNames;
    
    // Metadata
    private long inferenceTimeMs;
    private long timestamp;
    
    // Risk categorization
    private RiskCategory riskCategory;
    
    // Feature importance (optional, from SHAP)
    private Map<String, Double> featureImportance;
    
    public enum RiskCategory {
        LOW("Low Risk", 0.0, 0.25),
        MODERATE("Moderate Risk", 0.25, 0.50),
        HIGH("High Risk", 0.50, 0.75),
        CRITICAL("Critical Risk", 0.75, 1.0);
        
        private final String label;
        private final double min;
        private final double max;
        
        RiskCategory(String label, double min, double max) {
            this.label = label;
            this.min = min;
            this.max = max;
        }
        
        public static RiskCategory fromProbability(double probability) {
            for (RiskCategory category : values()) {
                if (probability >= category.min && probability < category.max) {
                    return category;
                }
            }
            return CRITICAL;  // >= 0.75
        }
    }
    
    /**
     * Get risk category based on prediction
     */
    public RiskCategory getRiskCategory() {
        if (riskCategory == null) {
            riskCategory = RiskCategory.fromProbability(prediction);
        }
        return riskCategory;
    }
}
```

package com.hospital.flink.ml.features;


import org.apache.flink.api.common.functions.RichMapFunction;
import org.apache.flink.configuration.Configuration;


import java.util.*;


/**
 * Extract ML features from enriched patient events
 * Combines Module 2 (vitals/labs) + Module 3 (protocols) + Module 4 (CEP/windows)
 */
public class FeatureExtractor extends RichMapFunction<EnrichedEvent, FeatureVector> {
    
    private transient FeatureDefinition featureDefinition;
    
    @Override
    public void open(Configuration parameters) throws Exception {
        // Load feature schema
        this.featureDefinition = FeatureDefinition.load();
    }
    
    @Override
    public FeatureVector map(EnrichedEvent event) throws Exception {
        
        // Extract features from multiple sources
        Map<String, Float> features = new HashMap<>();
        
        // ═══════════════════════════════════════════════════════
        // DEMOGRAPHIC FEATURES (5 features)
        // ═══════════════════════════════════════════════════════
        features.put("age", (float) event.getPatientState().getAge());
        features.put("gender_male", event.getPatientState().getGender().equals("M") ? 1.0f : 0.0f);
        features.put("bmi", (float) event.getPatientState().getBMI());
        features.put("is_icu", event.getPatientState().getRiskIndicators().isInICU() ? 1.0f : 0.0f);
        features.put("admission_source_er", 
            event.getPatientState().getAdmissionSource().equals("ER") ? 1.0f : 0.0f);
        
        // ═══════════════════════════════════════════════════════
        // VITAL SIGN FEATURES (12 features)
        // ═══════════════════════════════════════════════════════
        Map<String, Object> vitals = event.getPatientState().getLatestVitals();
        
        features.put("heart_rate", getFloatValue(vitals, "heartrate"));
        features.put("systolic_bp", getFloatValue(vitals, "systolicbp"));
        features.put("diastolic_bp", getFloatValue(vitals, "diastolicbp"));
        features.put("respiratory_rate", getFloatValue(vitals, "respiratoryrate"));
        features.put("temperature", getFloatValue(vitals, "temperature"));
        features.put("oxygen_saturation", getFloatValue(vitals, "oxygensaturation"));
        
        // Derived vital features
        features.put("mean_arterial_pressure", 
            (getFloatValue(vitals, "systolicbp") + 2 * getFloatValue(vitals, "diastolicbp")) / 3);
        features.put("pulse_pressure", 
            getFloatValue(vitals, "systolicbp") - getFloatValue(vitals, "diastolicbp"));
        features.put("shock_index", 
            getFloatValue(vitals, "heartrate") / getFloatValue(vitals, "systolicbp"));
        
        // Binary vital flags
        features.put("is_tachycardic", 
            event.getPatientState().getRiskIndicators().isTachycardia() ? 1.0f : 0.0f);
        features.put("is_hypotensive", 
            event.getPatientState().getRiskIndicators().isHypotension() ? 1.0f : 0.0f);
        features.put("is_hypoxic", 
            event.getPatientState().getRiskIndicators().isHypoxia() ? 1.0f : 0.0f);
        
        // ═══════════════════════════════════════════════════════
        // LAB VALUE FEATURES (15 features)
        // ═══════════════════════════════════════════════════════
        Map<String, LabResult> labs = event.getPatientState().getRecentLabs();
        
        features.put("lactate", getLabValue(labs, "2524-7", 2.0f));  // LOINC: Lactate
        features.put("creatinine", getLabValue(labs, "2160-0", 1.0f));  // LOINC: Creatinine
        features.put("bun", getLabValue(labs, "3094-0", 20.0f));  // LOINC: BUN
        features.put("sodium", getLabValue(labs, "2951-2", 140.0f));  // LOINC: Sodium
        features.put("potassium", getLabValue(labs, "2823-3", 4.0f));  // LOINC: Potassium
        features.put("wbc", getLabValue(labs, "6690-2", 10.0f));  // LOINC: WBC
        features.put("hemoglobin", getLabValue(labs, "718-7", 14.0f));  // LOINC: Hemoglobin
        features.put("platelet", getLabValue(labs, "777-3", 250.0f));  // LOINC: Platelets
        features.put("glucose", getLabValue(labs, "2339-0", 100.0f));  // LOINC: Glucose
        features.put("bilirubin", getLabValue(labs, "1975-2", 1.0f));  // LOINC: Bilirubin
        features.put("alt", getLabValue(labs, "1742-6", 30.0f));  // LOINC: ALT
        features.put("ast", getLabValue(labs, "1920-8", 30.0f));  // LOINC: AST
        features.put("troponin", getLabValue(labs, "42719-5", 0.01f));  // LOINC: Troponin
        features.put("bnp", getLabValue(labs, "30934-4", 100.0f));  // LOINC: BNP
        features.put("inr", getLabValue(labs, "6301-6", 1.0f));  // LOINC: INR
        
        // ═══════════════════════════════════════════════════════
        // CLINICAL SCORE FEATURES (5 features)
        // ═══════════════════════════════════════════════════════
        features.put("news2_score", (float) event.getPatientState().getNews2Score());
        features.put("qsofa_score", (float) event.getPatientState().getQsofaScore());
        features.put("sofa_score", (float) event.getPatientState().getSofaScore());
        features.put("apache_score", (float) event.getPatientState().getApacheScore());
        features.put("combined_acuity", (float) event.getPatientState().getCombinedAcuityScore());
        
        // ═══════════════════════════════════════════════════════
        // TEMPORAL FEATURES (10 features)
        // ═══════════════════════════════════════════════════════
        
        // Time since admission (hours)
        features.put("hours_since_admission", 
            (float) ((System.currentTimeMillis() - event.getPatientState().getAdmissionTime()) 
                    / (1000.0 * 3600.0)));
        
        // Time since last vital/lab update
        features.put("hours_since_last_vital", 
            (float) ((System.currentTimeMillis() - event.getPatientState().getLastVitalUpdate()) 
                    / (1000.0 * 3600.0)));
        features.put("hours_since_last_lab", 
            (float) ((System.currentTimeMillis() - event.getPatientState().getLastLabUpdate()) 
                    / (1000.0 * 3600.0)));
        
        // Trend indicators (from Module 4 windowed analytics)
        features.put("vital_trend_worsening", 
            event.getModule4Analytics() != null && 
            event.getModule4Analytics().getVitalTrend().equals("WORSENING") ? 1.0f : 0.0f);
        
        features.put("lab_trend_worsening", 
            event.getModule4Analytics() != null && 
            event.getModule4Analytics().getLabTrend().equals("WORSENING") ? 1.0f : 0.0f);
        
        // Rate of change features
        features.put("hr_change_per_hour", 
            calculateVitalChangeRate(event, "heartrate"));
        features.put("bp_change_per_hour", 
            calculateVitalChangeRate(event, "systolicbp"));
        features.put("lactate_change_per_hour", 
            calculateLabChangeRate(event, "2524-7"));
        
        // Hour of day (cyclic encoding)
        int hourOfDay = LocalDateTime.now().getHour();
        features.put("hour_sin", (float) Math.sin(2 * Math.PI * hourOfDay / 24));
        features.put("hour_cos", (float) Math.cos(2 * Math.PI * hourOfDay / 24));
        
        // ═══════════════════════════════════════════════════════
        // MEDICATION FEATURES (8 features)
        // ═══════════════════════════════════════════════════════
        Map<String, Medication> meds = event.getPatientState().getActiveMedications();
        
        features.put("on_vasopressors", 
            event.getPatientState().getRiskIndicators().isOnVasopressors() ? 1.0f : 0.0f);
        features.put("on_antibiotics", 
            hasAntibiotics(meds) ? 1.0f : 0.0f);
        features.put("on_anticoagulation", 
            event.getPatientState().getRiskIndicators().isOnAnticoagulation() ? 1.0f : 0.0f);
        features.put("on_sedation", 
            hasSedation(meds) ? 1.0f : 0.0f);
        features.put("medication_count", (float) meds.size());
        features.put("high_risk_med_count", (float) countHighRiskMeds(meds));
        features.put("recent_med_change", 
            event.getPatientState().getRiskIndicators().isRecentMedicationChange() ? 1.0f : 0.0f);
        features.put("polypharmacy", meds.size() >= 5 ? 1.0f : 0.0f);
        
        // ═══════════════════════════════════════════════════════
        // COMORBIDITY FEATURES (10 features)
        // ═══════════════════════════════════════════════════════
        features.put("has_diabetes", 
            event.getPatientState().getRiskIndicators().isHasDiabetes() ? 1.0f : 0.0f);
        features.put("has_ckd", 
            event.getPatientState().getRiskIndicators().isHasChronicKidneyDisease() ? 1.0f : 0.0f);
        features.put("has_heart_failure", 
            event.getPatientState().getRiskIndicators().isHasHeartFailure() ? 1.0f : 0.0f);
        features.put("has_copd", 
            event.getPatientState().getRiskIndicators().isHasCOPD() ? 1.0f : 0.0f);
        features.put("has_cancer", 
            event.getPatientState().getRiskIndicators().isHasCancer() ? 1.0f : 0.0f);
        features.put("immunocompromised", 
            event.getPatientState().getRiskIndicators().isImmunocompromised() ? 1.0f : 0.0f);
        features.put("post_operative", 
            event.getPatientState().getRiskIndicators().isPostOperative() ? 1.0f : 0.0f);
        features.put("charlson_index", (float) event.getPatientState().getCharlsonIndex());
        features.put("elixhauser_score", (float) event.getPatientState().getElixhauserScore());
        features.put("prior_admissions_1year", 
            (float) event.getPatientState().getPriorAdmissionsOneYear());
        
        // ═══════════════════════════════════════════════════════
        // CEP PATTERN FEATURES (from Module 4) (5 features)
        // ═══════════════════════════════════════════════════════
        if (event.getCepPatternMatches() != null) {
            features.put("sepsis_pattern_matched", 
                event.getCepPatternMatches().hasSepsisPattern() ? 1.0f : 0.0f);
            features.put("deterioration_pattern_matched", 
                event.getCepPatternMatches().hasDeteriorationPattern() ? 1.0f : 0.0f);
            features.put("aki_pattern_matched", 
                event.getCepPatternMatches().hasAKIPattern() ? 1.0f : 0.0f);
            features.put("pattern_confidence_max", 
                (float) event.getCepPatternMatches().getMaxConfidence());
            features.put("patterns_matched_count", 
                (float) event.getCepPatternMatches().getMatchCount());
        } else {
            features.put("sepsis_pattern_matched", 0.0f);
            features.put("deterioration_pattern_matched", 0.0f);
            features.put("aki_pattern_matched", 0.0f);
            features.put("pattern_confidence_max", 0.0f);
            features.put("patterns_matched_count", 0.0f);
        }
        
        // ═══════════════════════════════════════════════════════
        // TOTAL: 70 features
        // ═══════════════════════════════════════════════════════
        
        // Convert to feature vector
        return FeatureVector.builder()
            .patientId(event.getPatientId())
            .features(features)
            .featureNames(new ArrayList<>(features.keySet()))
            .timestamp(System.currentTimeMillis())
            .sourceEvent(event)
            .build();
    }
    
    // Helper methods
    private float getFloatValue(Map<String, Object> map, String key) {
        Object value = map.get(key);
        if (value == null) return 0.0f;
        if (value instanceof Number) {
            return ((Number) value).floatValue();
        }
        return 0.0f;
    }

```java
    private String patientId;

    private List<String> featureNames;
    private long timestamp;

    /**
     * Convert to float array for ML model input
     * Features must be in consistent order
     */
    public float[] toArray(List<String> orderedFeatureNames) {
        float[] array = new float[orderedFeatureNames.size()];
        
        for (int i = 0; i < orderedFeatureNames.size(); i++) {
            String featureName = orderedFeatureNames.get(i);
            array[i] = features.getOrDefault(featureName, 0.0f);
        }
        
        return array;
    }
    
    /**
     * Validate feature completeness
     */
    public ValidationResult validate() {
        int missingCount = 0;
        int nanCount = 0;
        int infCount = 0;
        List<String> missingFeatures = new ArrayList<>();
        
        for (String featureName : featureNames) {
            Float value = features.get(featureName);
            
            if (value == null) {
                missingCount++;
                missingFeatures.add(featureName);
            } else if (Float.isNaN(value)) {
                nanCount++;
            } else if (Float.isInfinite(value)) {
                infCount++;
            }
        }
        
        boolean isValid = missingCount == 0 && nanCount == 0 && infCount == 0;
        
        return ValidationResult.builder()
            .isValid(isValid)
            .totalFeatures(featureNames.size())
            .missingCount(missingCount)
            .nanCount(nanCount)
            .infCount(infCount)
            .missingFeatures(missingFeatures)
            .build();
    }
}
```

```java
            return "Low Sepsis Risk (<10%): Continue standard monitoring";
        } else if (probability < 0.25) {
            return "Moderate Sepsis Risk (10-25%): Monitor SIRS criteria, check lactate q6h";
        } else if (probability < 0.50) {

            return "Very High Sepsis Risk (>50%): Initiate sepsis bundle immediately, notify ICU";
        }

    private double estimateOnsetTime(double probability) {

            recommendations.add("Consider empiric antibiotics if source identified");
```

```java
package com.hospital.flink.ml;


import org.apache.flink.streaming.api.datastream.DataStream;
import org.apache.flink.streaming.api.environment.StreamExecutionEnvironment;


/**
 * Main Flink job integrating ML inference with Module 4 output
 */
public class Module5_MLInferenceJob {
    
    public static void main(String[] args) throws Exception {
        
        StreamExecutionEnvironment env = StreamExecutionEnvironment.getExecutionEnvironment();
        env.setParallelism(32);
        env.enableCheckpointing(60000);
        
        // ═══════════════════════════════════════════════════════
        // INPUT: Module 4 enriched events
        // ═══════════════════════════════════════════════════════
        
        DataStream<EnrichedEvent> enrichedEvents = env
            .fromSource(
                createKafkaSource("ehr-events-module4-output"),
                WatermarkStrategy
                    .<EnrichedEvent>forBoundedOutOfOrderness(Duration.ofMinutes(5))
                    .withTimestampAssigner((event, ts) -> event.getTimestamp()),
                "module4-input"
            )
            .keyBy(EnrichedEvent::getPatientId);
        
        // ═══════════════════════════════════════════════════════
        // FEATURE ENGINEERING
        // ═══════════════════════════════════════════════════════
        
        DataStream<FeatureVector> featureVectors = enrichedEvents
            .map(new FeatureExtractor())
            .name("feature-extraction");
        
        // ═══════════════════════════════════════════════════════
        // ML MODEL INFERENCE (Parallel streams)
        // ═══════════════════════════════════════════════════════
        
        // Model 1: Mortality Prediction
        DataStream<MLPrediction> mortalityPredictions = featureVectors
            .map(new MortalityPredictionModel())
            .name("mortality-prediction");
        
        // Model 2: Sepsis Onset Prediction
        DataStream<MLPrediction> sepsisPredictions = featureVectors
            .map(new SepsisOnsetPredictionModel())
            .name("sepsis-prediction");
        
        // Model 3: Readmission Risk
        DataStream<MLPrediction> readmissionPredictions = featureVectors
            .map(new ReadmissionRiskModel())
            .name("readmission-prediction");
        
        // Model 4: AKI Progression (optional)
        DataStream<MLPrediction> akiPredictions = featureVectors
            .map(new AKIProgressionModel())
            .name("aki-prediction");
        
        // ═══════════════════════════════════════════════════════
        // MODEL ENSEMBLE & CALIBRATION
        // ═══════════════════════════════════════════════════════
        
        // Combine predictions by patient
        DataStream<MLPredictionBundle> predictionBundles = mortalityPredictions
            .union(sepsisPredictions)
            .union(readmissionPredictions)
            .union(akiPredictions)
            .keyBy(pred -> pred.getFeatureVector().getPatientId())
            .window(TumblingProcessingTimeWindows.of(Time.seconds(10)))
            .aggregate(new PredictionAggregator())
            .name("prediction-ensemble");
        
        // ═══════════════════════════════════════════════════════
        // GENERATE ML-ENHANCED ALERTS
        // ═══════════════════════════════════════════════════════
        
        DataStream<MLAlert> mlAlerts = predictionBundles
            .flatMap(new MLAlertGenerator())
            .name("ml-alert-generation");
        
        // ═══════════════════════════════════════════════════════
        // INTEGRATE WITH MODULE 4 ALERTS
        // ═══════════════════════════════════════════════════════
        
        DataStream<ClinicalAlert> module4Alerts = env
            .fromSource(
                createKafkaSource("ehr-alerts-module4"),
                WatermarkStrategy.noWatermarks(),
                "module4-alerts"
            );
        
        DataStream<EnhancedAlert> enhancedAlerts = module4Alerts
            .connect(mlAlerts.keyBy(MLAlert::getPatientId))
            .process(new AlertEnhancementFunction())
            .name("alert-enhancement");
        
        // ═══════════════════════════════════════════════════════
        // OUTPUT SINKS
        // ═══════════════════════════════════════════════════════
        
        // ML predictions → Kafka
        predictionBundles.sinkTo(
            createKafkaSink("ehr-ml-predictions")
        ).name("ml-predictions-sink");
        
        // ML alerts → Kafka
        mlAlerts.sinkTo(
            createKafkaSink("ehr-ml-alerts")
        ).name("ml-alerts-sink");
        
        // Enhanced alerts → Kafka
        enhancedAlerts.sinkTo(
            createKafkaSink("ehr-alerts-final")
        ).name("enhanced-alerts-sink");
        
        // Model metrics → Prometheus
        mortalityPredictions
            .map(pred -> pred.getModel().getMetrics())
            .addSink(new PrometheusSink())
            .name("model-metrics-sink");
        
        // Execute
        env.execute("Module 5: ML Inference Pipeline");
    }
}
```
