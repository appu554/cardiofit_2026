🔧 Quick Generation Script (Optional)
If you want to batch-create templates:
#!/bin/bash
# generate-lab-templates.sh


tests=("glucose" "creatinine" "bun" "sodium" "potassium")
loinc=("2345-7" "2160-0" "3094-0" "2951-2" "2823-3")


for i in "${!tests[@]}"; do
  test="${tests[$i]}"
  code="${loinc[$i]}"
  
  cat > "lab-tests/chemistry/${test}.yaml" << EOF
testId: "LAB-${test^^}-001"
testName: "${test^}"
loincCode: "${code}"
category: "CHEMISTRY"
# TODO: Complete definition
EOF


  echo "Created ${test}.yaml"
done

💡 Tips for Rapid YAML Creation
1. Start with institutional lab manual - Copy reference ranges from your hospital's existing documentation
2. Use AI assistance - Ask ChatGPT/Claude to generate YAML from lab manual text
3. Validate with clinical team - Quick 15-min review with pathologist/lab director
4. Iterate later - Don't need perfect on Day 1, can refine in Phase 6
5. Focus on high-impact tests first - The 10 critical care labs cover 80% of sepsis/STEMI/respiratory protocols
