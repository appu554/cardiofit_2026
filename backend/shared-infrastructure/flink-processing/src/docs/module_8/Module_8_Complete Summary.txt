# 🎯 **Module 8: Read Model Projections & Storage Layer**


## Executive Summary


Module 8 is the **final consumption layer** that materializes Kafka event streams into **queryable, optimized storage systems**. It creates specialized read models tailored for different access patterns: transactional queries (PostgreSQL), full-text search (Elasticsearch), graph relationships (Neo4j), time-series analytics (ClickHouse), and clinical interoperability (FHIR Store).

## 🎯 **Storage Selection Matrix**


| Use Case | Storage | Why This Storage? | Access Pattern |
|----------|---------|-------------------|----------------|
| **Clinical Interoperability** | FHIR Store (PostgreSQL + HAPI FHIR) | Standard-compliant, RESTful API | Read: By resource ID, search by parameters<br>Write: Create/Update FHIR resources |
| **Full-Text Search** | Elasticsearch | Fast text search, faceted search, autocomplete | Read: Complex queries across all fields<br>Write: Index documents with analyzed fields |
| **Graph Relationships** | Neo4j | Relationship traversal, pattern matching | Read: Cypher queries (shortest path, relationships)<br>Write: Create nodes and edges |
| **Analytics (OLAP)** | ClickHouse | Columnar storage, fast aggregations | Read: Complex analytical queries<br>Write: Batch inserts, immutable |
| **UPS Read Model** | PostgreSQL | ACID transactions, relational queries | Read: Patient summary by ID<br>Write: UPSERT patient state |
| **Time-Series Vitals** | InfluxDB | Optimized for time-series, downsampling | Read: Range queries, aggregations<br>Write: High-frequency inserts |

## 📦 **Component 8A: FHIR Store Projector**


### **Purpose:**
- Store clinical data in FHIR R4 format for interoperability
- Enable FHIR REST API queries
- Publish to HIE (Health Information Exchange)


### **Schema: HAPI FHIR Tables**


```sql
-- HAPI FHIR uses ~100 tables, key ones:


-- Resource storage
CREATE TABLE hfj_resource (
    res_id BIGSERIAL PRIMARY KEY,
    res_type VARCHAR(40) NOT NULL,        -- Patient, Observation, etc.
    res_version VARCHAR(7),
    has_tags BOOLEAN,
    res_deleted_at TIMESTAMP,
    res_published TIMESTAMP,
    res_updated TIMESTAMP,
    fhir_id VARCHAR(100),                 -- FHIR resource ID
    res_text BYTEA,                       -- Narrative text
    res_text_vc TEXT                      -- Text as string
);


-- Resource content (JSONB)
CREATE TABLE hfj_res_ver (
    pid BIGSERIAL PRIMARY KEY,
    res_id BIGINT NOT NULL,
    res_type VARCHAR(40) NOT NULL,
    res_version VARCHAR(7),
    has_tags BOOLEAN,
    res_encoding VARCHAR(5),
    res_text BYTEA,                       -- FHIR JSON content
    res_ver BIGINT NOT NULL,
    FOREIGN KEY (res_id) REFERENCES hfj_resource(res_id)
);


-- Search parameters (for FHIR search API)
CREATE TABLE hfj_spidx_string (
    sp_id BIGSERIAL PRIMARY KEY,
    res_id BIGINT NOT NULL,
    sp_name VARCHAR(100) NOT NULL,        -- Search parameter name
    sp_value_normalized VARCHAR(200),     -- Normalized value
    sp_value_exact VARCHAR(200),
    FOREIGN KEY (res_id) REFERENCES hfj_resource(res_id)
);


-- Indexes
CREATE INDEX idx_res_type ON hfj_resource(res_type);
CREATE INDEX idx_fhir_id ON hfj_resource(fhir_id);
CREATE INDEX idx_sp_string_name_value ON hfj_spidx_string(sp_name, sp_value_normalized);
```

### **Implementation: FHIR Store Projector**


```java
package com.hospital.module8.projectors;


import ca.uhn.fhir.context.FhirContext;
import ca.uhn.fhir.rest.client.api.IGenericClient;
import org.apache.kafka.streams.KafkaStreams;
import org.apache.kafka.streams.StreamsBuilder;
import org.apache.kafka.streams.kstream.KStream;
import org.hl7.fhir.r4.model.*;
import org.springframework.stereotype.Service;


/**
 * FHIR Store Projector
 * Consumes clinical events and projects to FHIR R4 resources
 */
@Service
public class FHIRStoreProjector {
    
    private final FhirContext fhirContext;
    private final IGenericClient fhirClient;
    private final FHIRResourceMapper resourceMapper;
    
    public FHIRStoreProjector() {
        this.fhirContext = FhirContext.forR4();
        this.fhirClient = fhirContext.newRestfulGenericClient("http://localhost:8080/fhir");
        this.resourceMapper = new FHIRResourceMapper();
    }
    
    /**
     * Start Kafka Streams processor
     */
    public void start() {
        StreamsBuilder builder = new StreamsBuilder();
        
        // Consume from multiple topics
        KStream<String, EnrichedEvent> enrichedEvents = 
            builder.stream("enriched-events");
        KStream<String, MLPrediction> mlPredictions = 
            builder.stream("ml-predictions");
        KStream<String, CEPAlert> cepAlerts = 
            builder.stream("cep-alerts");
        
        // Project enriched events to FHIR resources
        enrichedEvents.foreach((key, event) -> projectToFHIR(event));
        
        // Project ML predictions to FHIR RiskAssessment
        mlPredictions.foreach((key, prediction) -> projectMLPrediction(prediction));
        
        // Project CEP alerts to FHIR DetectedIssue
        cepAlerts.foreach((key, alert) -> projectCEPAlert(alert));
        
        // Start streams
        KafkaStreams streams = new KafkaStreams(builder.build(), getStreamsConfig());
        streams.start();
    }
    
    /**
     * Project clinical event to appropriate FHIR resource
     */
    private void projectToFHIR(EnrichedEvent event) {
        try {
            switch (event.getEventType()) {
                case "VITAL_SIGNS":
                    Observation vitalObs = resourceMapper.toVitalSignObservation(event);
                    upsertFHIRResource(vitalObs);
                    break;
                    
                case "LAB_RESULT":
                    Observation labObs = resourceMapper.toLabObservation(event);
                    upsertFHIRResource(labObs);
                    break;
                    
                case "MEDICATION_ORDER":
                    MedicationRequest medReq = resourceMapper.toMedicationRequest(event);
                    upsertFHIRResource(medReq);
                    break;
                    
                case "MEDICATION_ADMINISTRATION":
                    MedicationAdministration medAdmin = 
                        resourceMapper.toMedicationAdministration(event);
                    upsertFHIRResource(medAdmin);
                    break;
                    
                case "DIAGNOSIS":
                    Condition condition = resourceMapper.toCondition(event);
                    upsertFHIRResource(condition);
                    break;
                    
                case "PROCEDURE":
                    Procedure procedure = resourceMapper.toProcedure(event);
                    upsertFHIRResource(procedure);
                    break;
                    
                case "ENCOUNTER":
                    Encounter encounter = resourceMapper.toEncounter(event);
                    upsertFHIRResource(encounter);
                    break;
                    
                default:
                    log.warn("Unknown event type: {}", event.getEventType());
            }
        } catch (Exception e) {
            log.error("Failed to project event to FHIR: {}", event, e);
            sendToDeadLetterQueue(event, e);
        }
    }
    
    /**
     * Project ML prediction to FHIR RiskAssessment
     */
    private void projectMLPrediction(MLPrediction prediction) {
        RiskAssessment riskAssessment = new RiskAssessment();
        
        // Set patient reference
        riskAssessment.setSubject(new Reference("Patient/" + prediction.getPatientId()));
        
        // Set status
        riskAssessment.setStatus(RiskAssessment.RiskAssessmentStatus.FINAL);
        
        // Set method (ML model)
        CodeableConcept method = new CodeableConcept();
        method.setText("Machine Learning Model: " + prediction.getModelType());
        riskAssessment.setMethod(method);
        
        // Set prediction
        RiskAssessment.RiskAssessmentPredictionComponent predictionComp = 
            new RiskAssessment.RiskAssessmentPredictionComponent();
        
        // Outcome (e.g., "30-day mortality")
        CodeableConcept outcome = new CodeableConcept();
        outcome.setText(getOutcomeText(prediction.getModelType()));
        predictionComp.setOutcome(outcome);
        
        // Probability
        DecimalType probability = new DecimalType(prediction.getPrediction());
        predictionComp.setProbability(probability);
        
        // Risk category
        CodeableConcept qualitativeRisk = new CodeableConcept();
        qualitativeRisk.setText(prediction.getRiskCategory().toString());
        predictionComp.setQualitativeRisk(qualitativeRisk);
        
        riskAssessment.addPrediction(predictionComp);
        
        // Set occurrence time
        riskAssessment.setOccurrence(new DateTimeType(new Date(prediction.getTimestamp())));
        
        // Add note with explanation
        Annotation note = new Annotation();
        note.setText("Confidence: " + prediction.getConfidence() + 
                     "\nInference time: " + prediction.getInferenceTimeMs() + "ms");
        riskAssessment.addNote(note);
        
        // Store to FHIR server
        upsertFHIRResource(riskAssessment);
    }
    
    /**
     * Project CEP alert to FHIR DetectedIssue
     */
    private void projectCEPAlert(CEPAlert alert) {
        DetectedIssue issue = new DetectedIssue();
        
        // Set patient reference
        issue.setPatient(new Reference("Patient/" + alert.getPatientId()));
        
        // Set status
        issue.setStatus(DetectedIssue.DetectedIssueStatus.FINAL);
        
        // Set severity
        issue.setSeverity(mapSeverity(alert.getSeverity()));
        
        // Set code (alert type)
        CodeableConcept code = new CodeableConcept();
        code.setText(alert.getAlertType());
        issue.setCode(code);
        
        // Set detail
        issue.setDetail(alert.getMessage());
        
        // Set identified time
        issue.setIdentified(new DateTimeType(new Date(alert.getTimestamp())));
        
        // Add evidence (triggering events)
        if (alert.getTriggeringEvents() != null) {
            alert.getTriggeringEvents().forEach(event -> {
                Reference evidenceRef = new Reference();
                evidenceRef.setDisplay(event.toString());
                issue.addEvidence().addDetail(evidenceRef);
            });
        }
        
        // Store to FHIR server
        upsertFHIRResource(issue);
    }
    
    /**
     * Upsert FHIR resource (create or update)
     */
    private <T extends Resource> void upsertFHIRResource(T resource) {
        try {
            // Check if resource exists
            String resourceId = resource.getIdElement().getIdPart();
            
            if (resourceId != null) {
                // Update existing
                MethodOutcome outcome = fhirClient.update()
                    .resource(resource)
                    .execute();
                
                log.debug("Updated FHIR resource: {} (version: {})",
                    outcome.getId(),
                    outcome.getId().getVersionIdPart());
            } else {
                // Create new
                MethodOutcome outcome = fhirClient.create()
                    .resource(resource)
                    .execute();
                
                log.debug("Created FHIR resource: {}", outcome.getId());
            }
            
        } catch (Exception e) {
            log.error("Failed to upsert FHIR resource: {}", resource, e);
            throw e;
        }
    }
    
    /**
     * Map severity to FHIR DetectedIssueSeverity
     */
    private DetectedIssue.DetectedIssueSeverity mapSeverity(String severity) {
        switch (severity) {
            case "CRITICAL":
            case "HIGH":
                return DetectedIssue.DetectedIssueSeverity.HIGH;
            case "MODERATE":
                return DetectedIssue.DetectedIssueSeverity.MODERATE;
            default:
                return DetectedIssue.DetectedIssueSeverity.LOW;
        }
    }
    
    private String getOutcomeText(ModelType modelType) {
        switch (modelType) {
            case MORTALITY_PREDICTION:
                return "30-day mortality";
            case SEPSIS_ONSET:
                return "Sepsis onset within 6 hours";
            case READMISSION_RISK:
                return "30-day readmission";
            case AKI_PROGRESSION:
                return "Acute Kidney Injury progression";
            default:
                return "Unknown outcome";
        }
    }
}
```

### **FHIR Resource Mapper**


```java
package com.hospital.module8.mappers;


import org.hl7.fhir.r4.model.*;
import java.util.Date;


/**
 * Maps internal events to FHIR R4 resources
 */
public class FHIRResourceMapper {
    
    /**
     * Map vital signs event to FHIR Observation
     */
    public Observation toVitalSignObservation(EnrichedEvent event) {
        Observation observation = new Observation();
        
        // Set status
        observation.setStatus(Observation.ObservationStatus.FINAL);
        
        // Set category (vital-signs)
        observation.addCategory()
            .addCoding()
            .setSystem("http://terminology.hl7.org/CodeSystem/observation-category")
            .setCode("vital-signs")
            .setDisplay("Vital Signs");
        
        // Set code (LOINC)
        CodeableConcept code = new CodeableConcept();
        Coding coding = mapVitalTypeToLOINC(event.getDataType());
        code.addCoding(coding);
        observation.setCode(code);
        
        // Set subject (patient)
        observation.setSubject(new Reference("Patient/" + event.getPatientId()));
        
        // Set effective time
        observation.setEffective(new DateTimeType(new Date(event.getTimestamp())));
        
        // Set value
        Quantity value = new Quantity();
        value.setValue(event.getValue());
        value.setUnit(event.getUnit());
        value.setSystem("http://unitsofmeasure.org");
        value.setCode(event.getUnit());
        observation.setValue(value);
        
        // Add interpretation (if abnormal)
        if (event.getAbnormalFlag() != null) {
            observation.addInterpretation()
                .addCoding()
                .setSystem("http://terminology.hl7.org/CodeSystem/v3-ObservationInterpretation")
                .setCode(mapAbnormalFlag(event.getAbnormalFlag()))
                .setDisplay(event.getAbnormalFlag());
        }
        
        return observation;
    }
    
    /**
     * Map lab result to FHIR Observation
     */
    public Observation toLabObservation(EnrichedEvent event) {
        Observation observation = new Observation();
        
        observation.setStatus(Observation.ObservationStatus.FINAL);
        
        // Set category (laboratory)
        observation.addCategory()
            .addCoding()
            .setSystem("http://terminology.hl7.org/CodeSystem/observation-category")
            .setCode("laboratory")
            .setDisplay("Laboratory");
        
        // Set code (LOINC)
        CodeableConcept code = new CodeableConcept();
        code.addCoding()
            .setSystem("http://loinc.org")
            .setCode(event.getLoincCode())
            .setDisplay(event.getLabName());
        observation.setCode(code);
        
        // Set subject
        observation.setSubject(new Reference("Patient/" + event.getPatientId()));
        
        // Set effective time
        observation.setEffective(new DateTimeType(new Date(event.getTimestamp())));
        
        // Set value
        Quantity value = new Quantity();
        value.setValue(event.getValue());
        value.setUnit(event.getUnit());
        observation.setValue(value);
        
        // Add reference range
        if (event.getReferenceRange() != null) {
            Observation.ObservationReferenceRangeComponent refRange = 
                new Observation.ObservationReferenceRangeComponent();
            refRange.setLow(new Quantity().setValue(event.getReferenceRange().getLow()));
            refRange.setHigh(new Quantity().setValue(event.getReferenceRange().getHigh()));
            observation.addReferenceRange(refRange);
        }
        
        return observation;
    }
    
    /**
     * Map medication order to FHIR MedicationRequest
     */
    public MedicationRequest toMedicationRequest(EnrichedEvent event) {
        MedicationRequest medReq = new MedicationRequest();
        
        medReq.setStatus(MedicationRequest.MedicationRequestStatus.ACTIVE);
        medReq.setIntent(MedicationRequest.MedicationRequestIntent.ORDER);
        
        // Set medication
        CodeableConcept medication = new CodeableConcept();
        medication.addCoding()
            .setSystem("http://www.nlm.nih.gov/research/umls/rxnorm")
            .setCode(event.getRxNormCode())
            .setDisplay(event.getMedicationName());
        medReq.setMedication(medication);
        
        // Set subject
        medReq.setSubject(new Reference("Patient/" + event.getPatientId()));
        
        // Set authored time
        medReq.setAuthoredOn(new Date(event.getTimestamp()));
        
        // Set dosage instruction
        Dosage dosage = new Dosage();
        dosage.setText(event.getDosageText());
        dosage.setRoute(new CodeableConcept().setText(event.getRoute()));
        dosage.setTiming(new Timing().setRepeat(
            new Timing.TimingRepeatComponent()
                .setFrequency(event.getFrequency())
                .setPeriod(1)
                .setPeriodUnit(Timing.UnitsOfTime.D)
        ));
        
        Dosage.DosageDoseAndRateComponent doseAndRate = 
            new Dosage.DosageDoseAndRateComponent();
        doseAndRate.setDose(new Quantity()
            .setValue(event.getDose())
            .setUnit(event.getDoseUnit()));
        dosage.addDoseAndRate(doseAndRate);
        
        medReq.addDosageInstruction(dosage);
        
        return medReq;
    }
    
    /**
     * Map medication administration to FHIR MedicationAdministration
     */
    public MedicationAdministration toMedicationAdministration(EnrichedEvent event) {
        MedicationAdministration medAdmin = new MedicationAdministration();
        
        medAdmin.setStatus(MedicationAdministration.MedicationAdministrationStatus.COMPLETED);
        
        // Set medication
        CodeableConcept medication = new CodeableConcept();
        medication.setText(event.getMedicationName());
        medAdmin.setMedication(medication);
        
        // Set subject
        medAdmin.setSubject(new Reference("Patient/" + event.getPatientId()));
        
        // Set effective time
        medAdmin.setEffective(new DateTimeType(new Date(event.getTimestamp())));
        
        // Set dosage
        MedicationAdministration.MedicationAdministrationDosageComponent dosage = 
            new MedicationAdministration.MedicationAdministrationDosageComponent();
        dosage.setDose(new Quantity()
            .setValue(event.getDose())
            .setUnit(event.getDoseUnit()));
        dosage.setRoute(new CodeableConcept().setText(event.getRoute()));
        medAdmin.setDosage(dosage);
        
        return medAdmin;
    }
    
    /**
     * Map diagnosis to FHIR Condition
     */
    public Condition toCondition(EnrichedEvent event) {
        Condition condition = new Condition();
        
        condition.setClinicalStatus(new CodeableConcept()
            .addCoding()
            .setSystem("http://terminology.hl7.org/CodeSystem/condition-clinical")
            .setCode("active"));
        
        // Set code (ICD-10)
        CodeableConcept code = new CodeableConcept();
        code.addCoding()
            .setSystem("http://hl7.org/fhir/sid/icd-10")
            .setCode(event.getIcd10Code())
            .setDisplay(event.getDiagnosisName());
        condition.setCode(code);
        
        // Set subject
        condition.setSubject(new Reference("Patient/" + event.getPatientId()));
        
        // Set onset
        condition.setOnset(new DateTimeType(new Date(event.getOnsetDate())));
        
        return condition;
    }
    
    /**
     * Map encounter to FHIR Encounter
     */
    public Encounter toEncounter(EnrichedEvent event) {
        Encounter encounter = new Encounter();
        
        encounter.setStatus(Encounter.EncounterStatus.INPROGRESS);
        
        // Set class (inpatient, outpatient, emergency)
        encounter.setClass_(new Coding()
            .setSystem("http://terminology.hl7.org/CodeSystem/v3-ActCode")
            .setCode(event.getEncounterType())
            .setDisplay(event.getEncounterType()));
        
        // Set subject
        encounter.setSubject(new Reference("Patient/" + event.getPatientId()));
        
        // Set period
        Period period = new Period();
        period.setStart(new Date(event.getAdmissionTime()));
        if (event.getDischargeTime() != null) {
            period.setEnd(new Date(event.getDischargeTime()));
            encounter.setStatus(Encounter.EncounterStatus.FINISHED);
        }
        encounter.setPeriod(period);
        
        return encounter;
    }
    
    /**
     * Map vital type to LOINC code
     */
    private Coding mapVitalTypeToLOINC(String vitalType) {
        Coding coding = new Coding().setSystem("http://loinc.org");
        
        switch (vitalType.toLowerCase()) {
            case "heartrate":
                return coding.setCode("8867-4").setDisplay("Heart rate");
            case "systolicbp":
                return coding.setCode("8480-6").setDisplay("Systolic blood pressure");
            case "diastolicbp":
                return coding.setCode("8462-4").setDisplay("Diastolic blood pressure");
            case "respiratoryrate":
                return coding.setCode("9279-1").setDisplay("Respiratory rate");
            case "temperature":
                return coding.setCode("8310-5").setDisplay("Body temperature");
            case "oxygensaturation":
                return coding.setCode("2708-6").setDisplay("Oxygen saturation");
            default:
                return coding.setCode("unknown").setDisplay(vitalType);
        }
    }
    
    /**
     * Map abnormal flag to FHIR interpretation code
     */
    private String mapAbnormalFlag(String flag) {
        switch (flag) {
            case "HIGH":
                return "H";
            case "LOW":
                return "L";
            case "CRITICAL_HIGH":
                return "HH";
            case "CRITICAL_LOW":
                return "LL";
            default:
                return "N";
        }
    }
}
```

```java
package com.hospital.module8.projectors;


import co.elastic.clients.elasticsearch.ElasticsearchClient;
import co.elastic.clients.elasticsearch.core.IndexRequest;
import co.elastic.clients.elasticsearch.core.UpdateRequest;
import org.apache.kafka.streams.KafkaStreams;
import org.apache.kafka.streams.StreamsBuilder;
import org.apache.kafka.streams.kstream.KStream;
import org.springframework.stereotype.Service;


import java.io.IOException;
import java.time.Instant;
import java.util.*;


/**
 * Elasticsearch Projector
 * Indexes clinical data for full-text search
 */
@Service
public class ElasticsearchProjector {
    
    private final ElasticsearchClient esClient;
    
    public ElasticsearchProjector(ElasticsearchClient esClient) {
        this.esClient = esClient;
    }
    
    /**
     * Start Kafka Streams processor
     */
    public void start() {
        StreamsBuilder builder = new StreamsBuilder();
        
        // Consume enriched events
        KStream<String, EnrichedEvent> enrichedEvents = 
            builder.stream("enriched-events");
        
        // Consume ML predictions
        KStream<String, MLPrediction> mlPredictions = 
            builder.stream("ml-predictions");
        
        // Consume CEP alerts
        KStream<String, CEPAlert> cepAlerts = 
            builder.stream("cep-alerts");
        
        // Index enriched events
        enrichedEvents.foreach((key, event) -> indexClinicalEvent(event));
        
        // Update patient document with ML predictions
        mlPredictions
            .groupByKey()
            .reduce((oldValue, newValue) -> newValue)
            .toStream()
            .foreach((patientId, prediction) -> updatePatientMLPredictions(patientId, prediction));
        
        // Index alerts
        cepAlerts.foreach((key, alert) -> indexAlert(alert));
        
        // Start streams
        KafkaStreams streams = new KafkaStreams(builder.build(), getStreamsConfig());
        streams.start();
    }
    
    /**
     * Index clinical event
     */
    private void indexClinicalEvent(EnrichedEvent event) {
        try {
            Map<String, Object> document = buildClinicalEventDocument(event);
            
            IndexRequest<Map<String, Object>> request = IndexRequest.of(i -> i
                .index("clinical_events")
                .id(event.getEventId())
                .document(document)
            );
            
            esClient.index(request);
            
            // Also update patient document
            updatePatientDocument(event);
            
        } catch (IOException e) {
            log.error("Failed to index clinical event: {}", event, e);
            sendToDeadLetterQueue(event, e);
        }
    }
    
    /**
     * Build Elasticsearch document from clinical event
     */
    private Map<String, Object> buildClinicalEventDocument(EnrichedEvent event) {
        Map<String, Object> doc = new HashMap<>();
        
        doc.put("event_id", event.getEventId());
        doc.put("event_type", event.getEventType());
        doc.put("patient_id", event.getPatientId());
        doc.put("timestamp", Instant.ofEpochMilli(event.getTimestamp()));
        doc.put("department", event.getDepartment());
        doc.put("user_id", event.getUserId());
        doc.put("source_system", event.getSourceSystem());
        
        // Event-specific data
        switch (event.getEventType()) {
            case "VITAL_SIGNS":
                doc.put("vital_signs", buildVitalSignsObject(event));
                break;
                
            case "LAB_RESULT":
                doc.put("lab_result", buildLabResultObject(event));
                break;
                
            case "MEDICATION_ORDER":
            case "MEDICATION_ADMINISTRATION":
                doc.put("medication", buildMedicationObject(event));
                break;
                
            case "NOTE":
                doc.put("free_text", event.getNoteText());
                break;
        }
        
        return doc;
    }
    
    /**
     * Update patient document (UPSERT)
     */
    private void updatePatientDocument(EnrichedEvent event) throws IOException {
        String patientId = event.getPatientId();
        
        // Build update script
        Map<String, Object> params = new HashMap<>();
        
        switch (event.getEventType()) {
            case "VITAL_SIGNS":
                // Update latest vitals
                params.put("vital_signs", buildVitalSignsObject(event));
                break;
                
            case "DIAGNOSIS":
                // Add to active problems
                Map<String, Object> problem = new HashMap<>();
                problem.put("condition", event.getConditionName());
                problem.put("icd10_code", event.getIcd10Code());
                problem.put("onset_date", Instant.ofEpochMilli(event.getOnsetDate()));
                
                params.put("new_problem", problem);
                break;
                
            case "MEDICATION_ORDER":
                // Add to active medications
                Map<String, Object> medication = new HashMap<>();
                medication.put("name", event.getMedicationName());
                medication.put("rxnorm_code", event.getRxNormCode());
                medication.put("dose", event.getDosageText());
                
                params.put("new_medication", medication);
                break;
        }
        
        if (!params.isEmpty()) {
            params.put("last_updated", Instant.now());
            
            String script = buildUpdateScript(event.getEventType());
            
            UpdateRequest<Map<String, Object>, Map<String, Object>> request = 
                UpdateRequest.of(u -> u
                    .index("patients")
                    .id(patientId)
                    .script(s -> s
                        .inline(i -> i
                            .source(script)
                            .params(params)
                        )
                    )
                    .upsert(buildPatientDocument(event))
                );
            
            esClient.update(request, Map.class);
        }
    }
    
    /**
     * Update patient ML predictions
     */
    private void updatePatientMLPredictions(String patientId, MLPrediction prediction) {
        try {
            Map<String, Object> mlPrediction = new HashMap<>();
            mlPrediction.put("model_type", prediction.getModelType().toString());
            mlPrediction.put("probability", prediction.getPrediction());
            mlPrediction.put("risk_category", prediction.getRiskCategory().toString());
            mlPrediction.put("timestamp", Instant.ofEpochMilli(prediction.getTimestamp()));
            
            String script = """
                if (ctx._source.ml_predictions == null) {
                    ctx._source.ml_predictions = [];
                }
                ctx._source.ml_predictions.removeIf(p -> 
                    p.model_type == params.ml_prediction.model_type
                );
                ctx._source.ml_predictions.add(params.ml_prediction);
                ctx._source.last_updated = params.last_updated;
            """;
            
            Map<String, Object> params = new HashMap<>();
            params.put("ml_prediction", mlPrediction);
            params.put("last_updated", Instant.now());
            
            UpdateRequest<Map<String, Object>, Map<String, Object>> request = 
                UpdateRequest.of(u -> u
                    .index("patients")
                    .id(patientId)
                    .script(s -> s
                        .inline(i -> i
                            .source(script)
                            .params(params)
                        )
                    )
                );
            
            esClient.update(request, Map.class);
            
        } catch (IOException e) {
            log.error("Failed to update patient ML predictions: {}", patientId, e);
        }
    }
    
    /**
     * Index alert
     */
    private void indexAlert(CEPAlert alert) {
        try {
            Map<String, Object> document = new HashMap<>();
            document.put("alert_id", alert.getAlertId());
            document.put("patient_id", alert.getPatientId());
            document.put("alert_type", alert.getAlertType());
            document.put("severity", alert.getSeverity());
            document.put("message", alert.getMessage());
            document.put("source", alert.getSource());
            document.put("confidence", alert.getConfidence());
            document.put("status", alert.getStatus());
            document.put("created_at", Instant.ofEpochMilli(alert.getCreatedAt()));
            document.put("department", alert.getDepartment());
            
            IndexRequest<Map<String, Object>> request = IndexRequest.of(i -> i
                .index("alerts")
                .id(alert.getAlertId())
                .document(document)
            );
            
            esClient.index(request);
            
        } catch (IOException e) {
            log.error("Failed to index alert: {}", alert, e);
        }
    }
    
    /**
     * Build update script based on event type
     */
    private String buildUpdateScript(String eventType) {
        switch (eventType) {
            case "VITAL_SIGNS":
                return """
                    ctx._source.vital_signs = params.vital_signs;
                    ctx._source.last_updated = params.last_updated;
                """;
                
            case "DIAGNOSIS":
                return """
                    if (ctx._source.active_problems == null) {
                        ctx._source.active_problems = [];
                    }
                    ctx._source.active_problems.removeIf(p -> 
                        p.icd10_code == params.new_problem.icd10_code
                    );
                    ctx._source.active_problems.add(params.new_problem);
                    ctx._source.last_updated = params.last_updated;
                """;
                
            case "MEDICATION_ORDER":
                return """
                    if (ctx._source.active_medications == null) {
                        ctx._source.active_medications = [];
                    }
                    ctx._source.active_medications.removeIf(m -> 
                        m.rxnorm_code == params.new_medication.rxnorm_code
                    );
                    ctx._source.active_medications.add(params.new_medication);
                    ctx._source.last_updated = params.last_updated;
                """;
                
            default:
                return "ctx._source.last_updated = params.last_updated;";
        }
    }
    
    /**
     * Build patient document for UPSERT
     */
    private Map<String, Object> buildPatientDocument(EnrichedEvent event) {
        Map<String, Object> doc = new HashMap<>();
        doc.put("patient_id", event.getPatientId());
        doc.put("mrn", event.getMrn());
        doc.put("name", event.getPatientName());
        doc.put("date_of_birth", event.getDateOfBirth());
        doc.put("gender", event.getGender());
        doc.put("department", event.getDepartment());
        doc.put("room", event.getRoom());
        doc.put("admission_date", Instant.ofEpochMilli(event.getAdmissionTime()));
        doc.put("active_problems", new ArrayList<>());
        doc.put("active_medications", new ArrayList<>());
        doc.put("allergies", new ArrayList<>());
        doc.put("risk_scores", new HashMap<>());
        doc.put("ml_predictions", new ArrayList<>());
        doc.put("last_updated", Instant.now());
        return doc;
    }
    
    private Map<String, Object> buildVitalSignsObject(EnrichedEvent event) {
        Map<String, Object> vitals = new HashMap<>();
        vitals.put("heart_rate", event.getHeartRate());
        vitals.put("systolic_bp", event.getSystolicBP());
        vitals.put("diastolic_bp", event.getDiastolicBP());
        vitals.put("respiratory_rate", event.getRespiratoryRate());
        vitals.put("temperature", event.getTemperature());

        return vitals;
    }

        Map<String, Object> lab = new HashMap<>();
        lab.put("loinc_code", event.getLoincCode());
        lab.put("lab_name", event.getLabName());
        lab.put("value", event.getValue());
        lab.put("unit", event.getUnit());
        lab.put("abnormal_flag", event.getAbnormalFlag());
        return lab;
    }

        Map<String, Object> med = new HashMap<>();

        med.put("rxnorm_code", event.getRxNormCode());
        med.put("dose", event.getDosageText());
        med.put("route", event.getRoute());
        med.put("frequency", event.getFrequency());
        return med;
    }
```

```java
package com.hospital.module8.search;


import co.elastic.clients.elasticsearch.ElasticsearchClient;

import java.util.stream.Collectors;


/**
 * Clinical Search Service
 * Provides search capabilities over Elasticsearch indices
 */
@Service
public class ClinicalSearchService {
    
    private final ElasticsearchClient esClient;
    
    public ClinicalSearchService(ElasticsearchClient esClient) {
        this.esClient = esClient;
    }
    
    /**
     * Search patients by name, MRN, or diagnosis
     */
    public List<PatientSearchResult> searchPatients(String query) throws IOException {
        SearchRequest request = SearchRequest.of(s -> s
            .index("patients")
            .query(q -> q
                .multiMatch(m -> m
                    .query(query)
                    .fields("name^3", "mrn^2", "active_problems.condition", "active_medications.name")
                    .type(MultiMatchQuery.Type.BestFields)
                    .fuzziness("AUTO")
                )
            )
            .size(20)
        );
        
        SearchResponse<PatientSearchResult> response = 
            esClient.search(request, PatientSearchResult.class);
        
        return response.hits().hits().stream()
            .map(Hit::source)
            .collect(Collectors.toList());
    }
    
    /**
     * Autocomplete medication names
     */
    public List<String> autocompleteMedications(String prefix) throws IOException {
        SearchRequest request = SearchRequest.of(s -> s
            .index("patients")
            .query(q -> q
                .prefix(p -> p
                    .field("active_medications.name")
                    .value(prefix)
                )
            )
            .aggregations("medication_names", a -> a
                .terms(t -> t
                    .field("active_medications.name.keyword")
                    .size(10)
                )
            )
            .size(0)
        );
        
        SearchResponse<Void> response = esClient.search(request, Void.class);
        
        return response.aggregations()
            .get("medication_names")
            .sterms()
            .buckets()
            .array()
            .stream()
            .map(b -> b.key().stringValue())
            .collect(Collectors.toList());
    }
    
    /**
     * Search clinical events by patient ID and date range
     */
    public List<ClinicalEvent> searchClinicalEvents(
        String patientId,
        long startTime,
        long endTime
    ) throws IOException {
        SearchRequest request = SearchRequest.of(s -> s
            .index("clinical_events")
            .query(q -> q
                .bool(b -> b
                    .must(m -> m
                        .term(t -> t
                            .field("patient_id")
                            .value(patientId)
                        )
                    )
                    .must(m -> m
                        .range(r -> r
                            .field("timestamp")
                            .gte(JsonData.of(startTime))
                            .lte(JsonData.of(endTime))
                        )
                    )
                )
            )
            .sort(so -> so
                .field(f -> f
                    .field("timestamp")
                    .order(SortOrder.Desc)
                )
            )
            .size(100)
        );
        
        SearchResponse<ClinicalEvent> response = 
            esClient.search(request, ClinicalEvent.class);
        
        return response.hits().hits().stream()
            .map(Hit::source)
            .collect(Collectors.toList());
    }
    
    /**
     * Find patients with high-risk ML predictions
     */
    public List<PatientSearchResult> findHighRiskPatients(
        String modelType,
        double minProbability
    ) throws IOException {
        SearchRequest request = SearchRequest.of(s -> s
            .index("patients")
            .query(q -> q
                .nested(n -> n
                    .path("ml_predictions")
                    .query(nq -> nq
                        .bool(b -> b
                            .must(m -> m
                                .term(t -> t
                                    .field("ml_predictions.model_type")
                                    .value(modelType)
                                )
                            )
                            .must(m -> m
                                .range(r -> r
                                    .field("ml_predictions.probability")
                                    .gte(JsonData.of(minProbability))
                                )
                            )
                        )
                    )
                )
            )
            .sort(so -> so
                .field(f -> f
                    .field("ml_predictions.probability")
                    .order(SortOrder.Desc)
                    .nested(n -> n
                        .path("ml_predictions")
                    )

            .size(50)
        );
        
        SearchResponse<PatientSearchResult> response = 
            esClient.search(request, PatientSearchResult.class);
        
        return response.hits().hits().stream()
            .map(Hit::source)
            .collect(Collectors.toList());
    }
    
    /**
     * Faceted search: Count patients by department and risk category
     */
    public Map<String, Map<String, Long>> getDepartmentRiskDistribution() throws IOException {
        SearchRequest request = SearchRequest.of(s -> s
            .index("patients")
            .query(q -> q.matchAll(m -> m))
            .aggregations("by_department", a -> a
                .terms(t -> t
                    .field("department")
                )
                .aggregations("by_risk", aa -> aa
                    .nested(n -> n
                        .path("ml_predictions")
                    )
                    .aggregations("risk_categories", aaa -> aaa
                        .terms(t -> t
                            .field("ml_predictions.risk_category")
                        )
                    )
                )
            )
            .size(0)
        );
        
        SearchResponse<Void> response = esClient.search(request, Void.class);
        
        // Parse aggregations
        Map<String, Map<String, Long>> result = new HashMap<>();
        
        response.aggregations()
            .get("by_department")
            .sterms()
            .buckets()
            .array()
            .forEach(deptBucket -> {
                String department = deptBucket.key().stringValue();
                Map<String, Long> riskCounts = new HashMap<>();
                
                deptBucket.aggregations()
                    .get("by_risk")
                    .nested()
                    .aggregations()
                    .get("risk_categories")
                    .sterms()
                    .buckets()
                    .array()
                    .forEach(riskBucket -> {
                        String riskCategory = riskBucket.key().stringValue();
                        riskCounts.put(riskCategory, riskBucket.docCount());
                    });
                
                result.put(department, riskCounts);
            });
        
        return result;
    }
```

package com.hospital.module8.projectors;


import org.neo4j.driver.*;
import org.apache.kafka.streams.KafkaStreams;
import org.apache.kafka.streams.StreamsBuilder;
import org.apache.kafka.streams.kstream.KStream;
import org.springframework.stereotype.Service;


import java.util.*;


/**
 * Neo4j Graph Projector
 * Builds knowledge graph from clinical events
 */
@Service
public class Neo4jGraphProjector {
    
    private final Driver neo4jDriver;
    
    public Neo4jGraphProjector(Driver neo4jDriver) {
        this.neo4jDriver = neo4jDriver;
    }
    
    /**
     * Start Kafka Streams processor
     */
    public void start() {
        StreamsBuilder builder = new StreamsBuilder();
        
        // Consume enriched events
        KStream<String, EnrichedEvent> enrichedEvents = 
            builder.stream("enriched-events");
        
        // Consume ML predictions
        KStream<String, MLPrediction> mlPredictions = 
            builder.stream("ml-predictions");
        
        // Project to graph
        enrichedEvents.foreach((key, event) -> projectToGraph(event));
        mlPredictions.foreach((key, prediction) -> projectMLPredictionToGraph(prediction));
        
        // Start streams
        KafkaStreams streams = new KafkaStreams(builder.build(), getStreamsConfig());
        streams.start();
    }
    
    /**
     * Project clinical event to graph
     */
    private void projectToGraph(EnrichedEvent event) {
        try (Session session = neo4jDriver.session()) {
            switch (event.getEventType()) {
                case "PATIENT_ADMISSION":
                    createOrUpdatePatient(session, event);
                    break;
                    
                case "DIAGNOSIS":
                    linkPatientToDisease(session, event);
                    break;
                    
                case "MEDICATION_ORDER":
                    linkPatientToMedication(session, event);
                    break;
                    
                case "LAB_RESULT":
                    linkPatientToLab(session, event);
                    break;
                    
                case "ALLERGY":
                    linkPatientToAllergen(session, event);
                    break;
            }
        } catch (Exception e) {
            log.error("Failed to project to Neo4j: {}", event, e);
        }
    }
    
    /**
     * Create or update patient node
     */
    private void createOrUpdatePatient(Session session, EnrichedEvent event) {
        String query = """
            MERGE (p:Patient {id: $patient_id})
            SET p.mrn = $mrn,
                p.name = $name,
                p.date_of_birth = datetime($dob),
                p.gender = $gender,
                p.department = $department,

```java
package com.hospital.module8.graph;


import org.neo4j.driver.*;
import org.springframework.stereotype.Service;


import java.util.*;
import java.util.stream.Collectors;


/**
 * Graph Query Service
 * Executes complex graph queries for clinical insights

@Service
public class GraphQueryService {

    /**
     * Find all drug-drug interactions for a patient
     */
    public List<DrugInteraction> findDrugInteractions(String patientId) {
        String query = """
            MATCH (p:Patient {id: $patient_id})-[:TAKING_MEDICATION]->(m1:Medication)
            MATCH (p)-[:TAKING_MEDICATION]->(m2:Medication)
            WHERE m1 <> m2
            MATCH (m1)-[i:INTERACTS_WITH]->(m2)
            RETURN 
                m1.name AS drug1,
                m2.name AS drug2,
                i.severity AS severity,
                i.mechanism AS mechanism,
                i.clinical_significance AS significance,
                i.management AS management
            ORDER BY 
                CASE i.severity
                    WHEN 'SEVERE' THEN 1
                    WHEN 'MODERATE' THEN 2
                    ELSE 3
                END
        """;
        
        try (Session session = neo4jDriver.session()) {
            Result result = session.run(query, Map.of("patient_id", patientId));
            
            return result.list(record -> 
                DrugInteraction.builder()
                    .drug1(record.get("drug1").asString())
                    .drug2(record.get("drug2").asString())
                    .severity(record.get("severity").asString())
                    .mechanism(record.get("mechanism").asString())
                    .clinicalSignificance(record.get("significance").asString())
                    .management(record.get("management").asString())
                    .build()
            );
        }
    }
    
    /**
     * Find contraindications for a patient
     */
    public List<Contraindication> findContraindications(String patientId) {
        String query = """
            MATCH (p:Patient {id: $patient_id})-[:TAKING_MEDICATION]->(m:Medication)
            MATCH (p)-[:HAS_DIAGNOSIS]->(d:Disease)
            MATCH (m)-[c:CONTRAINDICATED_IN]->(d)
            RETURN 
                m.name AS medication,
                d.name AS disease,
                c.severity AS severity,
                c.rationale AS rationale,
                c.alternative_options AS alternatives
            ORDER BY 
                CASE c.severity
                    WHEN 'ABSOLUTE' THEN 1
                    WHEN 'RELATIVE' THEN 2
                    ELSE 3
                END
        """;
        
        try (Session session = neo4jDriver.session()) {
            Result result = session.run(query, Map.of("patient_id", patientId));
            
            return result.list(record -> 
                Contraindication.builder()
                    .medication(record.get("medication").asString())
                    .disease(record.get("disease").asString())
                    .severity(record.get("severity").asString())
                    .rationale(record.get("rationale").asString())
                    .alternatives(record.get("alternatives").asString())
                    .build()
            );
        }

    /**
     * Find required lab monitoring for patient's medications
     */
    public List<MonitoringRequirement> findMonitoringRequirements(String patientId) {

            MATCH (p:Patient {id: $patient_id})-[:TAKING_MEDICATION]->(m:Medication)
            MATCH (m)-[r:REQUIRES_MONITORING]->(l:Lab)

    /**
     * Find medication alternatives for a contraindication
     */
    public List<MedicationAlternative> findMedicationAlternatives(
        String medicationRxnorm,
        String patientId
    ) {
        String query = """
            MATCH (original:Medication {rxnorm: $rxnorm})
            MATCH (p:Patient {id: $patient_id})-[:HAS_DIAGNOSIS]->(d:Disease)
            MATCH (original)-[:TREATS]->(indication:Disease)
            
            // Find alternative medications for same indication
            MATCH (alt:Medication)-[:TREATS]->(indication)
            WHERE alt <> original
            
            // Ensure alternative is NOT contraindicated
            OPTIONAL MATCH contraindicationPath = 
                (alt)-[:CONTRAINDICATED_IN]->(patientDisease:Disease)
                     <-[:HAS_DIAGNOSIS]-(p)
            WHERE contraindicationPath IS NULL
            
            // Ensure no cross-reactive allergies
            OPTIONAL MATCH (p)-[:ALLERGIC_TO]->(allergen:Allergen)
            OPTIONAL MATCH allergyPath = 
                (alt)-[:CROSS_REACTS_WITH]->(allergen)
            WHERE allergyPath IS NULL
            
            RETURN 
                alt.name AS name,
                alt.rxnorm AS rxnorm,
                alt.class AS drug_class,
                alt.efficacy AS efficacy,
                alt.cost AS cost
            ORDER BY alt.efficacy DESC, alt.cost ASC
            LIMIT 5
        """;
        
        try (Session session = neo4jDriver.session()) {
            Result result = session.run(query, Map.of(
                "rxnorm", medicationRxnorm,
                "patient_id", patientId
            ));
            
            return result.list(record -> 
                MedicationAlternative.builder()
                    .name(record.get("name").asString())
                    .rxnorm(record.get("rxnorm").asString())
                    .drugClass(record.get("drug_class").asString())
                    .efficacy(record.get("efficacy").asDouble())
                    .cost(record.get("cost").asDouble())
                    .build()
            );
        }
    }
    
    /**
     * Find interaction chains (drug1 → drug2 → lab)
     */
    public List<InteractionChain> findInteractionChains(String patientId) {
        String query = """
            MATCH (p:Patient {id: $patient_id})-[:TAKING_MEDICATION]->(m1:Medication)
            MATCH path = (m1)-[:INTERACTS_WITH*1..2]-(m2:Medication)
            WHERE (p)-[:TAKING_MEDICATION]->(m2)
            
            // Find required monitoring
            OPTIONAL MATCH (m1)-[:INTERACTION_REQUIRES_MONITORING]->(l:Lab)
            
            RETURN 
                [node IN nodes(path) | node.name] AS chain,
                [rel IN relationships(path) | {
                    type: type(rel),
                    severity: rel.severity,
                    mechanism: rel.mechanism
                }] AS interactions,
                COLLECT(DISTINCT {
                    lab: l.name,
                    loinc: l.loinc,
                    frequency: l.frequency
                }) AS monitoring
            ORDER BY SIZE(chain) ASC
        """;
        
        try (Session session = neo4jDriver.session()) {
            Result result = session.run(query, Map.of("patient_id", patientId));
            
            return result.list(record -> 
                InteractionChain.builder()
                    .medicationChain(record.get("chain").asList(Value::asString))
                    .interactions(parseInteractions(record.get("interactions").asList()))
                    .monitoring(parseMonitoring(record.get("monitoring").asList()))
                    .build()
            );
        }
    }
    
    /**
     * Find patients with similar clinical profiles
     */
    public List<String> findSimilarPatients(String patientId, int limit) {
        String query = """
            MATCH (p1:Patient {id: $patient_id})-[:HAS_DIAGNOSIS]->(d:Disease)
            MATCH (p2:Patient)-[:HAS_DIAGNOSIS]->(d)
            WHERE p1 <> p2
            WITH p1, p2, COUNT(d) AS shared_diseases
            
            MATCH (p1)-[:TAKING_MEDICATION]->(m:Medication)
            MATCH (p2)-[:TAKING_MEDICATION]->(m)
            WITH p1, p2, shared_diseases, COUNT(m) AS shared_medications
            
            RETURN 
                p2.id AS patient_id,
                shared_diseases,
                shared_medications,
                (shared_diseases * 2 + shared_medications) AS similarity_score
            ORDER BY similarity_score DESC
            LIMIT $limit
        """;
        
        try (Session session = neo4jDriver.session()) {
            Result result = session.run(query, Map.of(
                "patient_id", patientId,
                "limit", limit
            ));
            
            return result.list(record -> record.get("patient_id").asString());
        }
    }
    
    /**
     * Shortest path between two medical concepts
     */
    public List<String> findShortestPath(String concept1, String concept2) {
        String query = """
            MATCH (n1), (n2)
            WHERE n1.name = $concept1 AND n2.name = $concept2
            MATCH path = shortestPath((n1)-[*]-(n2))
            RETURN [node IN nodes(path) | node.name] AS path
        """;
        
        try (Session session = neo4jDriver.session()) {
            Result result = session.run(query, Map.of(
                "concept1", concept1,
                "concept2", concept2
            ));
            
            if (result.hasNext()) {
                return result.single().get("path").asList(Value::asString);
            }
            return Collections.emptyList();
        }
    }
```

```java
    private String getShift(long timestamp) {
        LocalTime time = LocalDateTime.ofInstant(
            Instant.ofEpochMilli(timestamp), 
            ZoneId.systemDefault()
        ).toLocalTime();
        
        if (time.isBefore(LocalTime.of(7, 0))) {
            return "NIGHT";
        } else if (time.isBefore(LocalTime.of(15, 0))) {
            return "DAY";
        } else if (time.isBefore(LocalTime.of(23, 0))) {
            return "EVENING";
        } else {
            return "NIGHT";
        }
    }
    
    private String getDayOfWeek(long timestamp) {
        return LocalDateTime.ofInstant(
            Instant.ofEpochMilli(timestamp), 
            ZoneId.systemDefault()
        ).getDayOfWeek().toString();
    }
```

```java
import com.influxdb.client.WriteApiBlocking;
import com.influxdb.client.domain.WritePrecision;
import com.influxdb.client.write.Point;
import org.apache.kafka.streams.KafkaStreams;
import org.apache.kafka.streams.StreamsBuilder;
import org.apache.kafka.streams.kstream.KStream;
import org.springframework.stereotype.Service;


import java.time.Instant;
import java.util.*;


/**
 * InfluxDB Projector
 * Stores high-frequency time-series data
 */
@Service
public class InfluxDBProjector {
    
    private final InfluxDBClient influxClient;
    private final WriteApiBlocking writeApi;
    
    public InfluxDBProjector(InfluxDBClient influxClient) {
        this.influxClient = influxClient;
        this.writeApi = influxClient.getWriteApiBlocking();
    }
    
    /**
     * Start Kafka Streams processor
     */
    public void start() {
        StreamsBuilder builder = new StreamsBuilder();
        
        // Consume enriched events (vital signs only)
        KStream<String, EnrichedEvent> enrichedEvents = 
            builder.stream("enriched-events");
        
        // Filter for vital signs
        enrichedEvents
            .filter((key, event) -> event.getEventType().equals("VITAL_SIGNS"))
            .foreach((key, event) -> writeVitalSigns(event));
        
        // Start streams
        KafkaStreams streams = new KafkaStreams(builder.build(), getStreamsConfig());
        streams.start();
        
        // Setup downsampling tasks
        setupDownsamplingTasks();
    }
    
    /**
     * Write vital signs to InfluxDB
     */
    private void writeVitalSigns(EnrichedEvent event) {
        try {
            List<Point> points = new ArrayList<>();
            
            // Heart Rate
            if (event.getHeartRate() != null) {
                points.add(createPoint("vitals", "heart_rate", 
                    event.getHeartRate(), event));
            }
            
            // Blood Pressure
            if (event.getSystolicBP() != null) {
                points.add(createPoint("vitals", "systolic_bp", 
                    event.getSystolicBP(), event));
            }
            if (event.getDiastolicBP() != null) {
                points.add(createPoint("vitals", "diastolic_bp", 
                    event.getDiastolicBP(), event));
            }
            
            // Respiratory Rate
            if (event.getRespiratoryRate() != null) {
                points.add(createPoint("vitals", "respiratory_rate", 
                    event.getRespiratoryRate(), event));
            }
            
            // Temperature
            if (event.getTemperature() != null) {
                points.add(createPoint("vitals", "temperature", 
                    event.getTemperature(), event));
            }
            
            // Oxygen Saturation
            if (event.getOxygenSaturation() != null) {
                points.add(createPoint("vitals", "oxygen_saturation", 
                    event.getOxygenSaturation(), event));
            }
            
            // Write all points
            writeApi.writePoints("vitals_realtime", "hospital", points);
            
        } catch (Exception e) {
            log.error("Failed to write vital signs to InfluxDB: {}", event, e);
        }
    }
    
    /**
     * Create InfluxDB point
     */
    private Point createPoint(String measurement, String vitalType, 
                              Number value, EnrichedEvent event) {
        return Point.measurement(measurement)
            .addTag("patient_id", event.getPatientId())
            .addTag("department", event.getDepartment())
            .addTag("room", event.getRoom())
            .addTag("data_type", vitalType)
            .addField("value", value)
            .addField("quality_flag", event.getQualityFlag() != null ? 
                event.getQualityFlag() : "GOOD")
            .time(Instant.ofEpochMilli(event.getTimestamp()), WritePrecision.MS);
    }
    
    /**
     * Setup downsampling tasks (Flux queries)
     */
    private void setupDownsamplingTasks() {
        // Task 1: Downsample to 1-minute averages
        String downsample1Min = """
            option task = {name: "Downsample to 1min", every: 1m}
            
            from(bucket: "vitals_realtime")
              |> range(start: -2m)
              |> filter(fn: (r) => r["_measurement"] == "vitals")
              |> aggregateWindow(every: 1m, fn: mean, createEmpty: false)
              |> to(bucket: "vitals_1min")
        """;
        
        // Task 2: Downsample to 1-hour averages
        String downsample1Hour = """
            option task = {name: "Downsample to 1hour", every: 1h}
            
            from(bucket: "vitals_1min")
              |> range(start: -2h)
              |> filter(fn: (r) => r["_measurement"] == "vitals")
              |> aggregateWindow(every: 1h, fn: mean, createEmpty: false)
              |> to(bucket: "vitals_1hour")
        """;
        
        // These would be registered as InfluxDB tasks
        // (In production, create via InfluxDB UI or API)
    }
```
