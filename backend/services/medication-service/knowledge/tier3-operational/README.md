# TIER 3 - Operational Knowledge

## 🎯 **Purpose**
Real-world operational data that enhances clinical decisions with practical considerations.

## 📋 **Components**

### **Formulary & Cost Database (FCD)**
- Insurance coverage information
- Medication pricing data
- Prior authorization requirements
- Generic alternatives

### **Monitoring Requirements Database (MRD)**
- Lab monitoring schedules
- Critical value thresholds
- Safety surveillance protocols
- Alert configurations

## 🔗 **Integration Points**
- **Formulary Engine**: Uses FCD for cost optimization
- **Monitoring Engine**: Uses MRD for surveillance
- **Safety Engine**: Uses MRD for alert thresholds
- **Response Optimizer**: Uses both for enrichment

## 💡 **Value Add**
This tier transforms clinical recommendations into actionable, practical guidance:
- **Cost considerations**: Find covered alternatives
- **Monitoring plans**: Specific lab schedules
- **Safety alerts**: Proactive surveillance
- **Real-world constraints**: Insurance, availability

**Makes clinical intelligence practical and implementable.**
