# TIER 2 - Decision Support (THE BRAIN)

## 🎯 **Purpose**
The intelligent decision-making layer that determines what to do and what data is needed.

## 📋 **Components**

### **Orchestrator Rule Base (ORB) - THE BRAIN**
- Recipe selection logic
- Intelligent routing rules
- Priority matrices
- Exception handling

### **Context Service Recipe Book (CSRB)**
- Data gathering instructions
- Context requirements per medication
- Data source optimization
- Performance tuning

## 🧠 **Critical Importance**
This tier contains **THE BRAIN** of the system:
- **ORB**: Makes the critical decision of which recipe to use
- **CSRB**: Determines exactly what data is needed
- **Without this tier**: System becomes generic and unintelligent

## 🚨 **The ORB is the Most Critical Component**
The Orchestrator Rule Base is the **single most important** part of the entire architecture:
- It transforms generic requests into intelligent clinical decisions
- It generates Intent Manifests that drive the entire flow
- It embodies clinical expertise and best practices
- **If ORB fails, the entire system becomes generic**

**This tier is what makes the system clinically intelligent rather than just a calculator.**
