# TIER 1 - Core Clinical Knowledge

## 🎯 **Purpose**
Foundation clinical knowledge that drives all clinical decisions and calculations.

## 📋 **Components**

### **Medication Knowledge Core (MKC)**
- Drug encyclopedia with RxNorm codes
- Drug interactions database
- Contraindications and warnings
- Therapeutic classifications

### **Clinical Recipe Book (CRB)**
- Dosing algorithms and clinical procedures
- Safety validation protocols
- Monitoring requirements
- Evidence-based calculations

## 🚨 **Critical Importance**
This tier contains the **core clinical intelligence**. Without this knowledge:
- ORB cannot make intelligent routing decisions
- Rust engine cannot perform accurate calculations
- Safety validations cannot function
- Clinical rationale cannot be provided

**All files in this tier are REQUIRED for system operation.**
