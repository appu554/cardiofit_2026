# Medication Knowledge Core (MKC)

## 🎯 **Purpose**
Comprehensive drug encyclopedia that provides the foundation for all clinical decisions.

## 📋 **Files**

### **drug_encyclopedia.json**
- RxNorm codes and drug properties
- Pharmacokinetic parameters
- Therapeutic classifications
- Standard dosing information

### **drug_interactions.json**
- Drug-drug interactions database
- Severity classifications
- Interaction mechanisms
- Management recommendations

### **contraindications.json**
- Absolute and relative contraindications
- Patient condition mappings
- Risk assessments
- Alternative recommendations

### **therapeutic_classes.json**
- Drug classification system
- Therapeutic equivalents
- Class-specific protocols

## 🔗 **Integration Points**
- **ORB Engine**: Uses for intelligent routing decisions
- **Rust Recipe Engine**: Uses for calculation parameters
- **Safety Engine**: Uses for interaction checking
- **Context Planner**: Uses for data requirements

**This is the clinical intelligence foundation of the entire system.**
