# Final Clinical Logic Recipe Book - Complete Edition

## Overview
This represents the complete set of **Clinical Logic Recipes** that reside within domain services (Medication Service, Order Service, etc.) and contain the clinical validation rules, safety algorithms, and decision logic.

## Enhanced Recipe Structure (Final)

```yaml
Recipe:
  # Core Identity
  id: unique-identifier-v1.0
  name: Human-readable name
  description: Clinical purpose
  priority: 1-100 (higher = more critical)
  
  # Activation
  triggers:
    - condition sets that activate this recipe
  
  # Clinical Metadata
  clinicalRationale: Medical justification
  evidenceBase:
    guidelines: ["Source guideline", "Evidence level"]
    references: ["Key studies"]
  
  # Data Requirements (Reference to Context Recipe)
  contextRecipeRef: "context-recipe-id"
  contextRequirements: GraphQL-like data needs
  
  # Core Logic
  validationLogic:
    CAE: Clinical assessment engine rules
    ProtocolEngine: Protocol compliance checks
    GraphDB: Relationship-based validations
  
  # Decision Support
  clinicalDecisionSupport:
    explanation:
      provider: "Clear rationale"
      patient: "Patient-friendly explanation"
    alternatives:
      suggested: ["Alternative approaches"]
      rationale: ["Why suggested"]
  
  # Cost & Efficiency
  costConsiderations:
    enabled: true/false
    factors:
      drugCost: ["Generic alternatives"]
      resourceUtilization: ["Monitoring costs"]
  
  # ML Enhancement Points
  mlEnhancement:
    patternDetection: ["Unusual patterns to flag"]
    outcomeTracking: ["Metrics to monitor"]
    personalization: ["Adaptation points"]
  
  # Performance & Failure
  performanceHint: "optimization guidance"
  fallbackBehavior: "degraded mode operation"
  qosTier: "platinum|gold|silver|bronze"
```

    - Determine if AKI present (KDIGO criteria)
    - Apply Cockcroft-Gault for drugs with CG-based dosing
    - Adjust dose based on renal function per FDA guidelines
    - Flag if multiple nephrotoxins prescribed
    - Alert if recent contrast exposure + nephrotoxic drug
    - Check cumulative nephrotoxic burden score
mlEnhancement:
  patternDetection:
    - "Rapid eGFR decline patterns"
    - "High-risk drug combinations"
  outcomeTracking:
    - "AKI development post-prescription"
