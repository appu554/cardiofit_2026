IOR Confounder Scoring Enhancement — Implementation Plan
Clinical Rationale
Why the current confounder model is clinically dangerous:
The Intervention-Outcome Record (IOR) system is the platform's evidence engine. When a physician sees a decision card saying "Similar patients (n=43) on SGLT2i showed median HbA1c delta of -0.9 at 12 weeks," that evidence comes from IOR outcome records. The credibility of every similar-patient comparison, every therapy response validation, and every phenotype cluster pathway recommendation depends on whether the IOR outcome data accurately attributes changes to the intervention rather than to confounders.
The current ComputeConfounderScore in ior_generator.go computes a 0-1 score from two inputs: concurrent medication count (×0.15 per med, capped at 0.45) and adherence drop (×0.5, capped at 0.25). If there are 3 concurrent med changes and a 0.3 adherence drop, the confounder score is 0.45 + 0.15 = 0.60, and the outcome record gets ConfidenceLevel: "LOW".
This model misses every non-pharmacological confounder — which in cardiometabolic care are frequently the dominant confounders. Consider these real scenarios:
Scenario 1 — Ramadan attribution error: A Muslim diabetic patient in Hyderabad starts SGLT2i in February. The 12-week IOR checkpoint falls in April — during Ramadan. His HbA1c drops from 8.2 to 7.4 (delta -0.8). The IOR system attributes this to SGLT2i with confounder score 0.0 (no concurrent meds, no adherence drop), confidence HIGH. But Ramadan fasting dramatically alters glucose patterns — extended fasting produces lower mean glucose, and many patients have genuinely lower HbA1c during Ramadan that rebounds afterward (Salti et al., 2004, Diabetes Care; Beshyah et al., 2019, Diabetes Research and Clinical Practice). The -0.8 delta is partly SGLT2i and partly Ramadan. Without the Ramadan confounder, the IOR overstates SGLT2i efficacy for this patient, and the similar-patient query overstates expected benefit for future patients.
Scenario 2 — Seasonal HbA1c attribution error: An Australian patient in Melbourne starts metformin intensification in March (autumn). The 26-week checkpoint falls in September (end of winter). HbA1c drops from 7.8 to 7.2. The IOR attributes this to metformin. But Tseng et al. (2005, Diabetologia) and Gikas et al. (2009, European Journal of Clinical Investigation) demonstrated a 0.2-0.4% seasonal HbA1c variation — higher in winter, lower in summer in temperate climates (reversed in tropical climates where summer heat reduces activity). In Melbourne, autumn-to-winter would typically show HbA1c rising, not falling — so the metformin effect may actually be stronger than the delta suggests. In Chennai (tropical), the seasonal pattern reverses. Without seasonal adjustment, IOR data from temperate-climate patients and tropical-climate patients is pooled without correction, introducing systematic bias in similar-patient queries.
Scenario 3 — Sick-day contamination: A patient on ACEi + SGLT2i develops a UTI (common SGLT2i side effect). During the 2-week illness, they experience dehydration, temporarily elevated creatinine, and apparent eGFR decline. The 4-week IOR checkpoint captures eGFR 48 (down from baseline 55). Confounder score: 0.0 (no concurrent meds, no adherence drop). The IOR records DELTA_EGFR of -7, confidence HIGH. This is a sick-day artifact, not a real renal decline — eGFR would recover after illness resolution. Without sick-day detection, the outcome record permanently attributes renal decline to the ACEi + SGLT2i combination, poisoning future similar-patient queries for patients with comparable eGFR.
Scenario 4 — Festival season weight gain misattributed to drug failure: An Indian patient starts GLP-1 RA for weight management in September. The 12-week checkpoint falls in December — after Diwali (October/November), Navratri feasting, and the wedding season. Weight increased from 88kg to 91kg despite GLP-1 RA. Confounder score: 0.0. The IOR records the GLP-1 RA as having FAILED for weight management, confidence HIGH. But the 3kg gain is likely 2-3kg festival weight gain partially offset by 1-2kg of drug-driven weight loss. Without seasonal/cultural confounder detection, the similar-patient query shows "GLP-1 RA failed for weight management in this phenotype cluster" — a dangerously misleading conclusion that discourages appropriate prescribing.
Scenario 5 — Steroid course contamination: Patient with CKD + T2DM receives a 2-week prednisolone course for a gout flare. During those 2 weeks, FBG jumps from 140 to 220, HbA1c measured at the 4-week checkpoint shows 8.5 (up from 7.8). The IOR attributes this to the current glycaemic regimen "failing." In reality, the steroid-induced hyperglycaemia is transient and unrelated to the regimen's efficacy. This is the most common iatrogenic confounder in cardiometabolic care — steroids affect both glucose AND blood pressure dramatically.
________________


What the literature says about confounding in treatment effect estimation:
Hernán and Robins (2020, "Causal Inference: What If") establish that valid causal inference from observational data requires adjusting for time-varying confounders. In cardiometabolic care, the dominant time-varying confounders are:
1. Concurrent medication changes (partially captured in current model)
2. Adherence variation (partially captured)
3. Seasonal variation — HbA1c (Tseng 2005, Gikas 2009), BP (Barnett 2007, Stergiou 2002), weight (Visscher & Seidell 2004)
4. Religious/cultural fasting — Ramadan (IDF-DAR 2021), Navratri, Lent, Yom Kippur
5. Acute illness — infections, hospitalization, acute kidney injury episodes
6. Iatrogenic confounders — steroid courses, post-surgical state, chemotherapy
7. Lifestyle disruption — job change, bereavement, migration, financial stress
8. Pregnancy/lactation — dramatically alters glucose, BP, weight, and eGFR
9. Dietary shifts — festival seasons, vacation, diet program initiation
10. Exercise discontinuation/initiation — seasonal (monsoon reduces outdoor activity in India)
The current model captures #1 and #2. We need to capture #3-#10 to produce clinically trustworthy IOR evidence.
The statistical approach — propensity-score-adjacent confounder adjustment:
We're not doing full causal inference (that requires counterfactual estimation and is computationally expensive for real-time clinical use). Instead, we're computing a confounder burden score that:
* Quantifies how many confounders were active during the outcome measurement window
* Weights each confounder by its expected effect size on the outcome type
* Produces a 0-1 score where higher = more confounded = less attributable to the intervention
* Maps to a confidence level (HIGH/MODERATE/LOW) that downstream consumers use to weight the evidence
This is closer to a "quality of evidence" score than a formal causal adjustment. It doesn't correct the delta — it flags how much to trust it. A confounder score of 0.7 means "70% of the plausible confounding space is active — interpret this outcome with extreme caution." The similar-patient query can then filter or down-weight high-confounder outcomes.
________________


India-Specific Confounder Landscape:
India has the richest confounder calendar of any market due to the intersection of religious diversity, seasonal extremes, and cultural food practices:
* Ramadan (30 days): Dawn-to-sunset fasting. Glucose patterns dramatically altered. IDF-DAR 2021 guidelines specific to diabetic patients during Ramadan. Affects ~15% of India's patient population (200M+ Muslims). HbA1c may drop 0.3-0.5% during Ramadan and rebound 2-4 weeks after.
* Navratri (9 days, twice yearly): Many Hindu patients fast or restrict to vegetarian/sattvic diet. Can lower caloric intake significantly. Affects glucose and weight metrics.
* Diwali season (2-4 weeks): Sweets, festive meals, reduced exercise. Weight gain 1-3kg typical. HbA1c and FBG may rise 0.2-0.5%.
* Wedding season (October-February): Extended social eating. Duration makes this a stronger confounder than single festivals.
* Monsoon season (June-September): Outdoor exercise drops dramatically. Patients in Mumbai, Chennai, Kolkata may show seasonal deconditioning. BP may rise due to reduced physical activity.
* Summer heat (April-June): Dehydration can cause apparent eGFR decline (pre-renal azotemia), BP changes due to vasodilation. Patients in Rajasthan, central India affected most.
Australia-Specific Confounder Landscape:
* Seasonal variation (temperate): Melbourne, Sydney, Adelaide show winter HbA1c elevation of 0.2-0.3%. Perth and Brisbane show less variation. Darwin (tropical) shows minimal seasonal effect.
* Holiday season (December-January): Extended summer holidays, dietary changes, alcohol consumption. Weight gain and glucose deterioration common.
* Indigenous cultural events: Sorry Business (bereavement practices), ceremony season — may affect medication adherence and lifestyle for extended periods. Must be handled with cultural sensitivity — the system should flag as confounder without pathologizing cultural practice.
* Bushfire/flood season: Natural disasters disrupt medication supply, access to care, and create stress responses. The 2019-2020 bushfire season caused measurable population-level HbA1c increases in affected regions (McArthur et al., 2021).

Architecture Overview
The enhanced confounder scoring system extends the existing IOR infrastructure in KB-20 with three new components:
1. Confounder Calendar Service (KB-20): Loads and queries seasonal, religious, and cultural confounder windows from market-config YAML. For any given patient-date-market combination, returns active confounders with expected effect sizes.

2. Clinical Event Confounder Detector (KB-20): Scans the patient's recent clinical history for acute illness markers (elevated CRP, hospitalization events, steroid prescriptions, UTI diagnoses) that constitute iatrogenic or illness confounders.

3. Enhanced Confounder Scorer (KB-20): Replaces the current ComputeConfounderScore with a multi-factor scoring engine that integrates medication confounders (existing), calendar confounders (new), clinical event confounders (new), and lifestyle disruption signals (from engagement data).

Data flow:
IOR Generator (existing)
    │
    ├── Fetch concurrent meds (existing)
    ├── Fetch adherence data (existing)
    │
    ├── NEW: Query Confounder Calendar
    │     ├── Is this window during Ramadan? (+weight)
    │     ├── Is this window during Diwali season? (+weight)
    │     ├── Is this a seasonal high/low for this outcome type? (+weight)
    │     └── Is this a monsoon/heat period? (+weight)
    │
    ├── NEW: Query Clinical Event History
    │     ├── Any steroid course in window? (+weight)
    │     ├── Any hospitalization in window? (+weight)
    │     ├── Any acute illness markers? (+weight)
    │     └── Any sick-day events? (+weight)
    │
    ├── NEW: Query Engagement/Lifestyle Signals
    │     ├── Engagement collapse during window? (+weight)
    │     ├── Exercise pattern change? (+weight)
    │     └── Dietary pattern change (sodium, carbs)? (+weight)
    │
    └── Enhanced ComputeConfounderScore()
          ├── Medication subscore (existing, refined)
          ├── Calendar subscore (new)
          ├── Clinical event subscore (new)
          ├── Lifestyle subscore (new)
          ├── Composite score (0-1)
          ├── Confidence level (HIGH/MODERATE/LOW)
          └── Confounder narrative (human-readable explanation)


Critical design principle: The confounder score doesn't adjust the outcome delta. It provides a quality-of-evidence flag. A delta of -0.8 HbA1c with confounder score 0.7 still records delta -0.8, but with confidence LOW and a narrative explaining which confounders were active. Downstream consumers (similar-patient query, therapy response validation, card evidence chains) decide how to weight the evidence based on the confidence level.

// kb-20-patient-profile/internal/models/confounder.go
package models


import "time"


// ConfounderCategory classifies the type of confounder.
type ConfounderCategory string


const (
    ConfounderMedication     ConfounderCategory = "MEDICATION"
    ConfounderAdherence      ConfounderCategory = "ADHERENCE"
    ConfounderSeasonal       ConfounderCategory = "SEASONAL"
    ConfounderReligiousFast  ConfounderCategory = "RELIGIOUS_FASTING"
    ConfounderFestivalDiet   ConfounderCategory = "FESTIVAL_DIETARY"
    ConfounderAcuteIllness   ConfounderCategory = "ACUTE_ILLNESS"
    ConfounderIatrogenic     ConfounderCategory = "IATROGENIC"
    ConfounderLifestyle      ConfounderCategory = "LIFESTYLE"
    ConfounderEnvironmental  ConfounderCategory = "ENVIRONMENTAL"
)


// ConfounderFactor represents a single active confounder during an outcome window.
type ConfounderFactor struct {
    Category          ConfounderCategory `json:"category"`
    Name              string             `json:"name"`                // e.g., "RAMADAN_2026", "STEROID_COURSE", "MONSOON_SEASON"
    Weight            float64            `json:"weight"`              // 0-1 contribution to composite score
    AffectedOutcomes  []string           `json:"affected_outcomes"`   // DELTA_HBA1C, DELTA_SBP, DELTA_WEIGHT, DELTA_EGFR
    ExpectedDirection string             `json:"expected_direction"`  // INCREASES, DECREASES, VARIABLE
    ExpectedMagnitude string             `json:"expected_magnitude"`  // e.g., "0.2-0.4% HbA1c", "2-3 kg weight"
    WindowStart       time.Time          `json:"window_start"`
    WindowEnd         time.Time          `json:"window_end"`
    OverlapDays       int                `json:"overlap_days"`        // days overlapping with the outcome window
    OverlapPct        float64            `json:"overlap_pct"`         // % of outcome window affected
    Source            string             `json:"source"`              // CALENDAR, CLINICAL_EVENT, ENGAGEMENT, LAB
    Confidence        string             `json:"confidence"`          // HIGH (documented event), MODERATE (calendar), LOW (inferred)
}


// EnhancedConfounderResult is the full output of the enhanced confounder scorer.
type EnhancedConfounderResult struct {
    // Composite
    CompositeScore    float64            `json:"composite_score"`     // 0-1
    ConfidenceLevel   string             `json:"confidence_level"`    // HIGH (<0.2), MODERATE (0.2-0.5), LOW (>0.5)
    
    // Per-category subscores
    MedicationScore   float64            `json:"medication_score"`
    CalendarScore     float64            `json:"calendar_score"`
    ClinicalEventScore float64           `json:"clinical_event_score"`
    LifestyleScore    float64            `json:"lifestyle_score"`
    
    // Detailed factors
    ActiveFactors     []ConfounderFactor `json:"active_factors"`
    FactorCount       int                `json:"factor_count"`
    
    // Narrative
    Narrative         string             `json:"narrative"`           // human-readable summary
    
    // Recommendations
    ShouldDefer       bool               `json:"should_defer"`        // true if confounder load is so high that outcome should be deferred
    DeferReasonCode   string             `json:"defer_reason_code,omitempty"` // e.g., "RAMADAN_ACTIVE", "STEROID_WASHOUT"
    SuggestedRecheckWeeks int            `json:"suggested_recheck_weeks,omitempty"`
}


// CalendarEvent represents a seasonal/religious/cultural event in the confounder calendar.
type CalendarEvent struct {
    Name              string   `yaml:"name" json:"name"`
    Category          string   `yaml:"category" json:"category"` // RELIGIOUS_FASTING, FESTIVAL_DIETARY, SEASONAL, ENVIRONMENTAL
    RecurrenceType    string   `yaml:"recurrence_type" json:"recurrence_type"` // ISLAMIC_LUNAR, HINDU_LUNAR, GREGORIAN_FIXED, GREGORIAN_SEASONAL
    GregorianApprox   []int    `yaml:"gregorian_approx_month" json:"gregorian_approx_month,omitempty"` // for seasonal
    DurationDays      int      `yaml:"duration_days" json:"duration_days"`
    AffectedOutcomes  []string `yaml:"affected_outcomes" json:"affected_outcomes"`
    ExpectedDirection string   `yaml:"expected_direction" json:"expected_direction"` // DECREASES_GLUCOSE, INCREASES_GLUCOSE, INCREASES_WEIGHT, etc.
    ExpectedMagnitude string   `yaml:"expected_magnitude" json:"expected_magnitude"`
    BaseWeight        float64  `yaml:"base_weight" json:"base_weight"` // weight when fully overlapping
    PostEventWashout  int      `yaml:"post_event_washout_days" json:"post_event_washout_days"` // days after event end before effects clear
    AppliesTo         string   `yaml:"applies_to" json:"applies_to"` // ALL, MUSLIM, HINDU, CHRISTIAN
    Notes             string   `yaml:"notes" json:"notes,omitempty"`
}


// ClinicalEventConfounder represents an acute clinical event that confounds outcomes.
type ClinicalEventConfounder struct {
    EventType         string   `json:"event_type"`   // STEROID_COURSE, HOSPITALIZATION, ACUTE_INFECTION, SURGERY, PREGNANCY
    DetectionMethod   string   `json:"detection_method"` // LAB_MARKER, MEDICATION_RECORD, DIAGNOSIS_CODE, PATIENT_REPORTED
    AffectedOutcomes  []string `json:"affected_outcomes"`
    ExpectedDirection string   `json:"expected_direction"`
    Weight            float64  `json:"weight"`
    WashoutDays       int      `json:"washout_days"` // days after event resolution before outcomes are reliable
}


// OutcomeConfounderAnnotation extends the existing OutcomeRecord with confounder detail.
type OutcomeConfounderAnnotation struct {
    OutcomeID         string                  `json:"outcome_id"`
    ConfounderResult  EnhancedConfounderResult `json:"confounder_result"`
    AdjustedConfidence string                 `json:"adjusted_confidence"`
    OriginalDelta     float64                 `json:"original_delta"`
    // We do NOT adjust the delta — we annotate it
}

import (

type ConfounderCalendar struct {
    market            string
    religiousEvents   []calendarEventConfig
    festivalEvents    []calendarEventConfig
    seasonalEvents    []calendarEventConfig
    indigenousEvents  []calendarEventConfig
    seasonalHbA1c     seasonalHbA1cConfig
}


type calendarEventConfig struct {
    Name              string   `yaml:"name"`
    Category          string   `yaml:"category"`
    DurationDays      int      `yaml:"duration_days"`
    AffectedOutcomes  []string `yaml:"affected_outcomes"`
    ExpectedDirection string   `yaml:"expected_direction"`
    ExpectedMagnitude string   `yaml:"expected_magnitude"`
    BaseWeight        float64  `yaml:"base_weight"`
    PostEventWashout  int      `yaml:"post_event_washout_days"`
    AppliesTo         string   `yaml:"applies_to"`
    RegionSpecific    bool     `yaml:"region_specific"`
    Regions           []string `yaml:"regions"`
    GregorianMonths   []int    `yaml:"gregorian_approx_month"`
    Dates2026         struct {
        Start string `yaml:"start"`
        End   string `yaml:"end"`
    } `yaml:"dates_2026"`
    Dates2027 struct {
        Start string `yaml:"start"`
        End   string `yaml:"end"`
    } `yaml:"dates_2027"`
}


type seasonalHbA1cConfig struct {
    Pattern       string  `yaml:"pattern"`
    PeakMonths    []int   `yaml:"peak_months"`
    TroughMonths  []int   `yaml:"trough_months"`
    MagnitudePct  float64 `yaml:"magnitude_pct"`
}


type calendarYAML struct {
    ReligiousEvents  []calendarEventConfig `yaml:"religious_events"`
    FestivalEvents   []calendarEventConfig `yaml:"festival_events"`
    SeasonalEvents   []calendarEventConfig `yaml:"seasonal_events"`
    IndigenousEvents []calendarEventConfig `yaml:"indigenous_events"`
    SeasonalHbA1c    seasonalHbA1cConfig   `yaml:"seasonal_hba1c"`
}


func LoadConfounderCalendar(configDir, market string) (*ConfounderCalendar, error) {
    path := filepath.Join(configDir, market, "confounder_calendar.yaml")
    data, err := os.ReadFile(path)
    if err != nil {
        return nil, fmt.Errorf("loading confounder calendar for %s: %w", market, err)
    }


    var raw calendarYAML
    if err := yaml.Unmarshal(data, &raw); err != nil {
        return nil, fmt.Errorf("parsing confounder calendar: %w", err)
    }


    return &ConfounderCalendar{
        market:           market,
        religiousEvents:  raw.ReligiousEvents,
        festivalEvents:   raw.FestivalEvents,
        seasonalEvents:   raw.SeasonalEvents,
        indigenousEvents: raw.IndigenousEvents,
        seasonalHbA1c:    raw.SeasonalHbA1c,
    }, nil
}


// FindActiveConfounders returns all calendar-based confounders active during the given window.
func (c *ConfounderCalendar) FindActiveConfounders(
    windowStart, windowEnd time.Time,
    religiousAffiliation string,
    region string,
) []models.ConfounderFactor {
    var factors []models.ConfounderFactor


    allEvents := make([]calendarEventConfig, 0)
    allEvents = append(allEvents, c.religiousEvents...)
    allEvents = append(allEvents, c.festivalEvents...)
    allEvents = append(allEvents, c.seasonalEvents...)
    allEvents = append(allEvents, c.indigenousEvents...)


    for _, event := range allEvents {
        // Check religious applicability
        if event.AppliesTo != "ALL" && event.AppliesTo != religiousAffiliation {
            continue
        }


        // Check region applicability
        if event.RegionSpecific && !containsString(event.Regions, region) {
            continue
        }


        // Determine event window
        eventStart, eventEnd := c.resolveEventDates(event, windowStart.Year())
        if eventStart.IsZero() {
            continue
        }


        // Extend event end by washout period
        washoutEnd := eventEnd.Add(time.Duration(event.PostEventWashout) * 24 * time.Hour)


        // Check overlap with outcome window
        overlapStart := maxTime(windowStart, eventStart)
        overlapEnd := minTime(windowEnd, washoutEnd)


        if overlapStart.Before(overlapEnd) {
            overlapDays := int(overlapEnd.Sub(overlapStart).Hours() / 24)
            windowDays := int(windowEnd.Sub(windowStart).Hours() / 24)
            overlapPct := 0.0
            if windowDays > 0 {
                overlapPct = float64(overlapDays) / float64(windowDays) * 100
            }


            // Weight scales with overlap percentage
            weight := event.BaseWeight * math.Min(overlapPct/100.0, 1.0)


            factors = append(factors, models.ConfounderFactor{
                Category:          models.ConfounderCategory(event.Category),
                Name:              event.Name,
                Weight:            math.Round(weight*1000) / 1000,
                AffectedOutcomes:  event.AffectedOutcomes,
                ExpectedDirection: event.ExpectedDirection,
                ExpectedMagnitude: event.ExpectedMagnitude,
                WindowStart:       eventStart,
                WindowEnd:         washoutEnd,
                OverlapDays:       overlapDays,
                OverlapPct:        math.Round(overlapPct*10) / 10,
                Source:            "CALENDAR",
                Confidence:        "MODERATE",
            })
        }
    }


    return factors
}


func (c *ConfounderCalendar) resolveEventDates(event calendarEventConfig, year int) (time.Time, time.Time) {
    // Try exact dates first (for religious events with known dates)
    if year == 2026 && event.Dates2026.Start != "" {
        start, _ := time.Parse("2006-01-02", event.Dates2026.Start)
        end, _ := time.Parse("2006-01-02", event.Dates2026.End)
        if !start.IsZero() && !end.IsZero() {
            return start, end
        }
    }
    if year == 2027 && event.Dates2027.Start != "" {
        start, _ := time.Parse("2006-01-02", event.Dates2027.Start)
        end, _ := time.Parse("2006-01-02", event.Dates2027.End)
        if !start.IsZero() && !end.IsZero() {
            return start, end
        }
    }


    // Fall back to seasonal approximation
    if len(event.GregorianMonths) > 0 {
        startMonth := event.GregorianMonths[0]
        start := time.Date(year, time.Month(startMonth), 1, 0, 0, 0, 0, time.UTC)
        end := start.Add(time.Duration(event.DurationDays) * 24 * time.Hour)
        return start, end
    }


    return time.Time{}, time.Time{}
}


// GetSeasonalHbA1cAdjustment returns the expected seasonal HbA1c deviation for a given month.
// Positive = HbA1c expected to be higher than annual mean; negative = lower.
func (c *ConfounderCalendar) GetSeasonalHbA1cAdjustment(month int) float64 {
    for _, m := range c.seasonalHbA1c.PeakMonths {
        if m == month {
            return c.seasonalHbA1c.MagnitudePct
        }
    }
    for _, m := range c.seasonalHbA1c.TroughMonths {
        if m == month {
            return -c.seasonalHbA1c.MagnitudePct
        }
    }
    return 0.0
}


func maxTime(a, b time.Time) time.Time {
    if a.After(b) { return a }
    return b
}


func minTime(a, b time.Time) time.Time {
    if a.Before(b) { return a }
    return b
}


func containsString(slice []string, s string) bool {
    for _, v := range slice {
        if v == s { return true }
    }
    return false
}

type ConfounderWeights struct {
    SteroidWeight        float64
    SteroidWashoutDays   int
    HospWeight           float64
    HospWashoutDays      int
    InfectionWeight      float64
    InfectionWashoutDays int
    AKIWeight            float64

    SurgeryWashoutDays   int

func NewClinicalEventDetector(weights *ConfounderWeights) *ClinicalEventDetector {

// Steroid drug name patterns
var steroidPatterns = []string{
    "prednisolone", "prednisone", "dexamethasone", "methylprednisolone",
    "hydrocortisone", "cortisone", "betamethasone", "triamcinolone",
}


// Antibiotic patterns (signal acute infection)
var antibioticPatterns = []string{
    "amoxicillin", "azithromycin", "ciprofloxacin", "levofloxacin",
    "doxycycline", "cephalexin", "trimethoprim", "nitrofurantoin",
    "metronidazole", "clindamycin", "augmentin", "cefuroxime",
}


func (d *ClinicalEventDetector) DetectConfounders(
    events []PatientClinicalEvent,
    windowStart, windowEnd time.Time,
) []models.ConfounderFactor {
    var factors []models.ConfounderFactor


    var steroidStart, steroidEnd *time.Time
    var hasHospitalization bool
    var hospDate time.Time
    var hospDuration int
    var hasAntibiotic bool
    var antibioticDate time.Time
    var creatinineValues []struct {
        Value float64
        Date  time.Time
    }


    for _, event := range events {
        switch event.Type {
        case "MEDICATION_START":
            drugLower := strings.ToLower(event.DrugName)
            if matchesAny(drugLower, steroidPatterns) {
                t := event.Date
                steroidStart = &t
            }
            if matchesAny(drugLower, antibioticPatterns) {
                hasAntibiotic = true
                antibioticDate = event.Date
            }


        case "MEDICATION_STOP":
            drugLower := strings.ToLower(event.DrugName)
            if matchesAny(drugLower, steroidPatterns) {
                t := event.Date
                steroidEnd = &t
            }


        case "HOSPITALIZATION":
            hasHospitalization = true
            hospDate = event.Date
            hospDuration = event.Duration


        case "LAB_RESULT":
            if event.LabType == "CREATININE" {
                creatinineValues = append(creatinineValues, struct {
                    Value float64
                    Date  time.Time
                }{event.Value, event.Date})
            }
        }
    }


    // Steroid course detection
    if steroidStart != nil {
        end := windowEnd
        if steroidEnd != nil {
            end = *steroidEnd
        }
        washoutEnd := end.Add(time.Duration(d.weights.SteroidWashoutDays) * 24 * time.Hour)


        if overlaps(*steroidStart, washoutEnd, windowStart, windowEnd) {
            overlapDays := computeOverlapDays(*steroidStart, washoutEnd, windowStart, windowEnd)
            factors = append(factors, models.ConfounderFactor{
                Category:          models.ConfounderIatrogenic,
                Name:              "STEROID_COURSE",
                Weight:            d.weights.SteroidWeight,
                AffectedOutcomes:  []string{"DELTA_HBA1C", "DELTA_FBG", "DELTA_SBP", "DELTA_WEIGHT"},
                ExpectedDirection: "WORSENS_ALL",
                ExpectedMagnitude: "Steroids raise glucose 50-100%, raise BP 10-20 mmHg, cause fluid retention",
                WindowStart:       *steroidStart,
                WindowEnd:         washoutEnd,
                OverlapDays:       overlapDays,
                Source:            "CLINICAL_EVENT",
                Confidence:        "HIGH",
            })
        }
    }


    // Hospitalization detection
    if hasHospitalization {
        hospEnd := hospDate.Add(time.Duration(hospDuration) * 24 * time.Hour)
        washoutEnd := hospEnd.Add(time.Duration(d.weights.HospWashoutDays) * 24 * time.Hour)


        if overlaps(hospDate, washoutEnd, windowStart, windowEnd) {
            factors = append(factors, models.ConfounderFactor{
                Category:          models.ConfounderAcuteIllness,
                Name:              "HOSPITALIZATION",
                Weight:            d.weights.HospWeight,
                AffectedOutcomes:  []string{"DELTA_HBA1C", "DELTA_SBP", "DELTA_EGFR", "DELTA_WEIGHT"},
                ExpectedDirection: "VARIABLE",
                ExpectedMagnitude: "Hospitalization disrupts all metabolic parameters. 6-week recovery expected.",
                WindowStart:       hospDate,
                WindowEnd:         washoutEnd,
                OverlapDays:       computeOverlapDays(hospDate, washoutEnd, windowStart, windowEnd),
                Source:            "CLINICAL_EVENT",
                Confidence:        "HIGH",
            })
        }
    }


    // Antibiotic → acute infection proxy
    if hasAntibiotic {
        infectionEnd := antibioticDate.Add(time.Duration(14) * 24 * time.Hour) // assume 2-week course
        washoutEnd := infectionEnd.Add(time.Duration(d.weights.InfectionWashoutDays) * 24 * time.Hour)


        if overlaps(antibioticDate, washoutEnd, windowStart, windowEnd) {
            factors = append(factors, models.ConfounderFactor{
                Category:          models.ConfounderAcuteIllness,
                Name:              "ACUTE_INFECTION",
                Weight:            d.weights.InfectionWeight,
                AffectedOutcomes:  []string{"DELTA_HBA1C", "DELTA_FBG", "DELTA_EGFR"},
                ExpectedDirection: "WORSENS_GLUCOSE_AND_RENAL",
                ExpectedMagnitude: "Infection-driven stress hyperglycaemia + transient renal impairment",
                WindowStart:       antibioticDate,
                WindowEnd:         washoutEnd,
                OverlapDays:       computeOverlapDays(antibioticDate, washoutEnd, windowStart, windowEnd),
                Source:            "CLINICAL_EVENT",
                Confidence:        "MODERATE",
            })
        }
    }


    // AKI detection from creatinine spike
    if len(creatinineValues) >= 2 {
        // Find baseline (oldest) and peak (highest recent)
        var baseline, peak struct {
            Value float64
            Date  time.Time
        }
        for _, cv := range creatinineValues {
            if baseline.Date.IsZero() || cv.Date.Before(baseline.Date) {
                baseline = cv
            }
            if cv.Date.After(windowStart.Add(-90*24*time.Hour)) && cv.Value > peak.Value {
                peak = cv
            }
        }


        // AKI: creatinine increase ≥1.5× baseline (KDIGO definition)
        if baseline.Value > 0 && peak.Value >= baseline.Value*1.5 {
            washoutEnd := peak.Date.Add(time.Duration(d.weights.AKIWashoutDays) * 24 * time.Hour)


            if overlaps(peak.Date, washoutEnd, windowStart, windowEnd) {
                factors = append(factors, models.ConfounderFactor{
                    Category:          models.ConfounderAcuteIllness,
                    Name:              "ACUTE_KIDNEY_INJURY",
                    Weight:            d.weights.AKIWeight,
                    AffectedOutcomes:  []string{"DELTA_EGFR"},
                    ExpectedDirection: "WORSENS_RENAL",
                    ExpectedMagnitude: "AKI causes transient eGFR decline that may recover. Washout 90 days.",
                    WindowStart:       peak.Date,
                    WindowEnd:         washoutEnd,
                    OverlapDays:       computeOverlapDays(peak.Date, washoutEnd, windowStart, windowEnd),
                    Source:            "CLINICAL_EVENT",
                    Confidence:        "HIGH",
                })
            }
        }
    }


    return factors
}


func matchesAny(s string, patterns []string) bool {
    for _, p := range patterns {
        if strings.Contains(s, p) {
            return true
        }
    }
    return false
}


func overlaps(aStart, aEnd, bStart, bEnd time.Time) bool {
    return aStart.Before(bEnd) && bStart.Before(aEnd)
}


func computeOverlapDays(aStart, aEnd, bStart, bEnd time.Time) int {
    start := maxTime(aStart, bStart)
    end := minTime(aEnd, bEnd)
    if start.After(end) {
        return 0
    }
    return int(math.Ceil(end.Sub(start).Hours() / 24))
}

type EnhancedConfounderInput struct {
    // Medication confounders (existing)
    ConcurrentMedCount int
    AdherenceDrop      float64


    // Calendar confounders (new)
    CalendarFactors []models.ConfounderFactor


    // Clinical event confounders (new)
    ClinicalEventFactors []models.ConfounderFactor


    // Lifestyle confounders (new)
    LifestyleFactors []models.ConfounderFactor


    // Context
    OutcomeType    string // DELTA_HBA1C, DELTA_SBP, DELTA_EGFR, DELTA_WEIGHT
    DeferOnRamadan bool
    DeferOnSteroid bool
}


type EnhancedConfounderScorer struct{}


func NewEnhancedConfounderScorer() *EnhancedConfounderScorer {
    return &EnhancedConfounderScorer{}
}


func (s *EnhancedConfounderScorer) Compute(input EnhancedConfounderInput) models.EnhancedConfounderResult {
    result := models.EnhancedConfounderResult{}
    var narrativeParts []string


    // 1. Medication subscore (refined from original)
    medScore := math.Min(float64(input.ConcurrentMedCount)*0.12, 0.40)
    adhScore := math.Min(input.AdherenceDrop*1.0, 0.25) // 10% drop → 0.10 score
    result.MedicationScore = math.Min(medScore+adhScore, 0.45)


    if input.ConcurrentMedCount > 0 {
        narrativeParts = append(narrativeParts,
            fmt.Sprintf("%d concurrent medication change(s)", input.ConcurrentMedCount))
        result.ActiveFactors = append(result.ActiveFactors, models.ConfounderFactor{
            Category: models.ConfounderMedication,
            Name:     "CONCURRENT_MED_CHANGES",
            Weight:   medScore,
        })
    }
    if input.AdherenceDrop > 0.05 {
        narrativeParts = append(narrativeParts,
            fmt.Sprintf("adherence drop of %.0f%%", input.AdherenceDrop*100))
        result.ActiveFactors = append(result.ActiveFactors, models.ConfounderFactor{
            Category: models.ConfounderAdherence,
            Name:     "ADHERENCE_DROP",
            Weight:   adhScore,
        })
    }


    // 2. Calendar subscore — only count factors that affect this outcome type
    for _, factor := range input.CalendarFactors {
        if !affectsOutcome(factor.AffectedOutcomes, input.OutcomeType) {
            continue
        }
        result.CalendarScore += factor.Weight
        result.ActiveFactors = append(result.ActiveFactors, factor)
        narrativeParts = append(narrativeParts, factor.Name)


        // Deferral check
        if input.DeferOnRamadan && factor.Name == "RAMADAN" && factor.OverlapPct > 50 {
            result.ShouldDefer = true
            result.DeferReasonCode = "RAMADAN_ACTIVE"
            result.SuggestedRecheckWeeks = 6 // recheck 6 weeks after Ramadan ends
        }
    }
    result.CalendarScore = math.Min(result.CalendarScore, 0.30) // cap calendar contribution


    // 3. Clinical event subscore
    for _, factor := range input.ClinicalEventFactors {
        if !affectsOutcome(factor.AffectedOutcomes, input.OutcomeType) {
            continue
        }
        result.ClinicalEventScore += factor.Weight
        result.ActiveFactors = append(result.ActiveFactors, factor)
        narrativeParts = append(narrativeParts, factor.Name)


        // Steroid deferral
        if input.DeferOnSteroid && factor.Name == "STEROID_COURSE" {
            result.ShouldDefer = true
            result.DeferReasonCode = "STEROID_WASHOUT"
            result.SuggestedRecheckWeeks = 4
        }
    }
    result.ClinicalEventScore = math.Min(result.ClinicalEventScore, 0.45) // cap


    // 4. Lifestyle subscore
    for _, factor := range input.LifestyleFactors {
        result.LifestyleScore += factor.Weight
        result.ActiveFactors = append(result.ActiveFactors, factor)
        narrativeParts = append(narrativeParts, factor.Name)
    }
    result.LifestyleScore = math.Min(result.LifestyleScore, 0.20) // cap


    // 5. Composite
    result.CompositeScore = math.Min(
        result.MedicationScore+result.CalendarScore+result.ClinicalEventScore+result.LifestyleScore,
        1.0,
    )
    result.CompositeScore = math.Round(result.CompositeScore*100) / 100
    result.FactorCount = len(result.ActiveFactors)


    // 6. Confidence level
    if result.CompositeScore < 0.20 {
        result.ConfidenceLevel = "HIGH"
    } else if result.CompositeScore < 0.50 {
        result.ConfidenceLevel = "MODERATE"
    } else {
        result.ConfidenceLevel = "LOW"
    }


    // High clinical event score forces deferral consideration
    if result.ClinicalEventScore >= 0.35 {
        result.ShouldDefer = true
        if result.DeferReasonCode == "" {
            result.DeferReasonCode = "HIGH_CLINICAL_EVENT_CONFOUNDING"
            result.SuggestedRecheckWeeks = 6
        }
    }


    // 7. Narrative
    if len(narrativeParts) == 0 {
        result.Narrative = "No significant confounders detected during outcome measurement window."
    } else {
        result.Narrative = fmt.Sprintf("Outcome confidence %s (score %.2f). Active confounders: %s.",
            result.ConfidenceLevel, result.CompositeScore, strings.Join(narrativeParts, ", "))
    }


    return result
}


func affectsOutcome(affectedOutcomes []string, outcomeType string) bool {
    if outcomeType == "" || len(affectedOutcomes) == 0 {
        return true // if no outcome type specified, assume relevant
    }
    for _, ao := range affectedOutcomes {
        if ao == outcomeType {
            return true
        }
    }
    return false
}

package services


// FilterIORByConfidence filters or down-weights IOR outcomes based on confounder confidence.
// Used when building similar-patient evidence for decision cards.
func FilterIORByConfidence(outcomes []IOROutcomeResult, minConfidence string) []IOROutcomeResult {
    confidenceRank := map[string]int{"HIGH": 3, "MODERATE": 2, "LOW": 1}
    minRank := confidenceRank[minConfidence]
    if minRank == 0 {
        minRank = 1 // default to accepting everything
    }


    var filtered []IOROutcomeResult
    for _, o := range outcomes {
        rank := confidenceRank[o.ConfidenceLevel]
        if rank >= minRank {
            filtered = append(filtered, o)
        }
    }
    return filtered
}

    highPct := float64(highConfidenceCount) / float64(totalOutcomes) * 100


    if highPct >= 70 {
        return evidenceSummary + " (evidence quality: HIGH — majority of outcomes have minimal confounding)"
    }
    if highPct >= 40 {
        return evidenceSummary + " (evidence quality: MODERATE — some outcomes affected by confounders)"
    }
    return evidenceSummary + " (evidence quality: LOW — significant confounding in outcome data; interpret with caution)"
}
