## Lessons for Pipeline Improvement

1. **Page classifier needed**: A pre-extraction page classifier that tags pages as "front matter", "introduction", "methodology", "clinical content", or "references" would prevent the F channel from extracting non-clinical pages entirely
2. **F channel over-extraction**: NuExtract LLM treats every paragraph as potentially relevant — it needs domain-specific prompting or post-extraction filtering to skip introductory and methodology text
3. **Tier assignment gap**: All 3 spans were assigned T2 (Clinical Accuracy) when they should have been T3 (Informational) at most — the tier classifier lacks awareness that Introduction pages contain methodology, not clinical recommendations
4. **Personnel pattern recurrence**: This is the third page with personnel names incorrectly extracted (Pages 15, 16, 19) — a name/credential pattern filter would catch this consistently
