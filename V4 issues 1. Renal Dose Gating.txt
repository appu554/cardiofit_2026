Renal Dose Gating — Implementation Plan
Clinical Rationale
Why this is patient safety priority #1:
Cardiometabolic patients on multi-drug regimens face real harm when eGFR decline crosses drug-specific thresholds without triggering medication review. The failure modes are concrete:
* SGLT2i below eGFR 20: Loss of glycaemic and cardioprotective efficacy; canagliflozin specifically contraindicated. CREDENCE/DAPA-CKD trials excluded patients below this threshold. Continuing generates false therapeutic confidence.
* Metformin below eGFR 30: Lactic acidosis risk. FDA boxed warning. KDIGO 2024 mandates discontinuation. Between eGFR 30-45, dose must halve (max 1000mg/day). This is the single most common prescribing error in diabetic nephropathy — prevalence studies show 15-25% of patients on metformin have eGFR below safe thresholds.
* GLP-1 RA dose adjustment: Exenatide contraindicated below eGFR 30. Liraglutide/semaglutide/dulaglutide require no renal adjustment but gastroparesis risk increases with declining renal function — monitoring frequency must escalate.
* ACEi/ARB + potassium: Hyperkalaemia risk escalates below eGFR 45. KDIGO mandates potassium monitoring within 1 week of initiation and after each dose change when eGFR <45.
* MRA (spironolactone/eplerenone): Contraindicated below eGFR 30 with potassium >5.0. FIDELIO-DKD established finerenone's renal/CV benefit but with strict potassium gating.
* Thiazide → Loop diuretic switch: Thiazides lose efficacy below eGFR 30. Must switch to loop diuretics. Your conflict detector flags thiazide's glycaemic effect but misses this far more dangerous efficacy cliff.
* NSAID avoidance: Below eGFR 60, NSAIDs accelerate renal decline. Not a prescribed drug in your platform, but patient-reported OTC use (captured via PATIENT_REPORTED observation type) should trigger warnings.
eGFR trajectory matters as much as absolute value. A patient at eGFR 48 with a slope of -5 mL/min/year will cross the metformin threshold in ~3.6 years — but if slope is -12, they'll cross in 1.5 years. The gating system must incorporate eGFR trajectory (already partially available via KB-26 renal domain) to generate anticipatory alerts, not just reactive ones.
India-specific considerations: Generic metformin is the first-line cornerstone in >90% of Indian T2DM patients (RSSDI pathway). Renal dose gating for metformin is therefore not an edge case — it's the most frequently triggered gate. Additionally, SGLT2i uptake is accelerating rapidly in urban India (Jardiance/Forxiga) but renal monitoring infrastructure in tier-2/3 cities often means eGFR is checked only at annual review, not quarterly. The system must flag patients whose last eGFR is >90 days old AND who are on renal-sensitive medications.
Australia-specific considerations: PBS authority prescriptions for SGLT2i require documented eGFR. RACGP guidelines mandate eGFR + ACR at least annually for all diabetic patients, quarterly if eGFR <60 or ACR >30. Aboriginal and Torres Strait Islander populations have 3-4x higher rates of CKD and often present at later stages. The indigenous overrides must lower monitoring thresholds.

Architecture Overview
The renal dose gating system spans three services:
1. KB-20 (Patient Profile): Stores current medications with drug class, dose, start date. Stores latest eGFR, eGFR slope, potassium, ACR. Exposes medication-renal-status query.
2. KB-23 (Decision Cards): Houses the RenalDoseGate — the core safety engine that cross-references active medications against eGFR thresholds. Produces gating verdicts that block, warn, or modify card recommendations. Integrates with the existing CardBuilder pipeline.
3. KB-26 (Metabolic Digital Twin): Computes eGFR trajectory slope (partially exists in MHRI renal domain). Publishes RENAL_THRESHOLD_APPROACHING events when projected time-to-threshold < configurable horizon (default 6 months).
Data flow:
eGFR lab result → Module 1b (Flink) → KB-20 (stores lab)
                                     → KB-26 (updates renal domain, computes slope)
                                     → KB-26 publishes RENAL_THRESHOLD_APPROACHING if projected
KB-23 card generation:
  1. Fetch patient medications from KB-20
  2. Fetch renal status (eGFR, slope, potassium, ACR) from KB-20/KB-26
  3. Run RenalDoseGate → produces GatingVerdict per medication
  4. GatingVerdict feeds into CardBuilder:
     - CONTRAINDICATED → generate IMMEDIATE urgency card for discontinuation
     - DOSE_REDUCE → generate URGENT card for dose adjustment
     - MONITOR_ESCALATE → generate ROUTINE card for increased lab frequency
     - ANTICIPATORY_ALERT → generate SCHEDULED card for proactive review
     - CLEARED → no renal gating action needed
  5. GatingVerdict also BLOCKS recommendations:
     - If a four-pillar card recommends SGLT2i initiation but eGFR < 20 → BLOCK card, substitute alternative
     - If a card recommends metformin intensification but eGFR < 30 → BLOCK card, flag discontinuation instead


Critical safety invariant: The renal dose gate is a hard gate — it runs BEFORE the card builder's recommendation engine, and its CONTRAINDICATED/DOSE_REDUCE verdicts cannot be overridden by any downstream logic. This is not a suggestion layer; it is a safety interlock.

// kb-23-decision-cards/internal/models/renal_gating.go
package models


import "time"


// GatingVerdict is the renal safety classification for a single medication.
type GatingVerdict string


const (
        VerdictContraindicated   GatingVerdict = "CONTRAINDICATED"
        VerdictDoseReduce        GatingVerdict = "DOSE_REDUCE"
        VerdictMonitorEscalate   GatingVerdict = "MONITOR_ESCALATE"
        VerdictAnticipatory      GatingVerdict = "ANTICIPATORY_ALERT"
        VerdictCleared           GatingVerdict = "CLEARED"
        VerdictInsufficientData  GatingVerdict = "INSUFFICIENT_DATA"
)


// RenalDrugRule defines eGFR thresholds for a single drug class.
type RenalDrugRule struct {
        DrugClass             string  `yaml:"drug_class" json:"drug_class"`
        ContraindicatedBelow  float64 `yaml:"contraindicated_below" json:"contraindicated_below"`
        DoseReduceBelow       float64 `yaml:"dose_reduce_below" json:"dose_reduce_below"`
        MaxDoseReducedMg      float64 `yaml:"max_dose_reduced_mg" json:"max_dose_reduced_mg"`
        MonitorEscalateBelow  float64 `yaml:"monitor_escalate_below" json:"monitor_escalate_below"`
        RequiresPotassiumCheck bool   `yaml:"requires_potassium_check" json:"requires_potassium_check"`
        PotassiumContraAbove  float64 `yaml:"potassium_contra_above" json:"potassium_contra_above"`
        EfficacyCliffBelow    float64 `yaml:"efficacy_cliff_below" json:"efficacy_cliff_below"` // e.g., thiazide at eGFR 30
        SubstituteClass       string  `yaml:"substitute_class" json:"substitute_class"`          // e.g., THIAZIDE → LOOP_DIURETIC
        AnticipateMonths      int     `yaml:"anticipate_months" json:"anticipate_months"`        // lookahead for trajectory
        SourceGuideline       string  `yaml:"source_guideline" json:"source_guideline"`
}


// RenalStatus holds the patient's current renal state for gating decisions.
type RenalStatus struct {
        EGFR               float64    `json:"egfr"`
        EGFRSlope          float64    `json:"egfr_slope"`           // mL/min/1.73m²/year (negative = declining)
        EGFRMeasuredAt     time.Time  `json:"egfr_measured_at"`
        EGFRDataPoints     int        `json:"egfr_data_points"`     // readings in trailing 12 months
        Potassium          *float64   `json:"potassium,omitempty"`   // mEq/L, may be unavailable
        PotassiumMeasuredAt *time.Time `json:"potassium_measured_at,omitempty"`
        ACR                *float64   `json:"acr,omitempty"`         // mg/g
        CKDStage           string     `json:"ckd_stage"`            // G1, G2, G3a, G3b, G4, G5
        IsRapidDecliner    bool       `json:"is_rapid_decliner"`    // slope worse than -5 mL/min/year
}


// MedicationGatingResult is the full gating output for one medication.
type MedicationGatingResult struct {
        DrugClass           string        `json:"drug_class"`
        DrugName            string        `json:"drug_name,omitempty"`
        CurrentDoseMg       float64       `json:"current_dose_mg,omitempty"`
        Verdict             GatingVerdict `json:"verdict"`
        Reason              string        `json:"reason"`
        ClinicalAction      string        `json:"clinical_action"`
        MaxSafeDoseMg       *float64      `json:"max_safe_dose_mg,omitempty"`
        SubstituteClass     string        `json:"substitute_class,omitempty"`
        MonitoringRequired  []string      `json:"monitoring_required,omitempty"`
        MonitoringFrequency string        `json:"monitoring_frequency,omitempty"` // WEEKLY, MONTHLY, QUARTERLY
        TimeToThreshold     *float64      `json:"time_to_threshold_months,omitempty"` // projected months to next threshold
        SourceGuideline     string        `json:"source_guideline"`
        EGFR                float64       `json:"egfr_at_evaluation"`
        EvaluatedAt         time.Time     `json:"evaluated_at"`
}


// PatientGatingReport is the full renal safety report for a patient.
type PatientGatingReport struct {
        PatientID          string                   `json:"patient_id"`
        RenalStatus        RenalStatus              `json:"renal_status"`
        MedicationResults  []MedicationGatingResult  `json:"medication_results"`
        HasContraindicated bool                     `json:"has_contraindicated"`
        HasDoseReduce      bool                     `json:"has_dose_reduce"`
        StaleEGFR          bool                     `json:"stale_egfr"`
        StaleEGFRDays      int                      `json:"stale_egfr_days,omitempty"`
        OverallUrgency     string                   `json:"overall_urgency"`
        BlockedRecommendations []string             `json:"blocked_recommendations,omitempty"`
}

        StaleEGFR StaleEGFRConfig
        Accessibility map[string]string // drug_class → HIGH/MODERATE/LOW
}


type StaleEGFRConfig struct {
        WarningDays        int  `yaml:"warning_days"`
        CriticalDays       int  `yaml:"critical_days"`
        HardBlockOnCritical bool `yaml:"hard_block_on_critical"`
        MonitoringFrequency map[string]int `yaml:"monitoring_frequency"`
}


type renalRulesYAML struct {
        StaleEGFR StaleEGFRConfig `yaml:"stale_egfr"`
        DrugRules []models.RenalDrugRule `yaml:"drug_rules"`
}


type marketRenalOverrideYAML struct {
        StaleEGFR         *StaleEGFRConfig           `yaml:"stale_egfr"`
        DrugRuleOverrides []marketDrugOverride        `yaml:"drug_rule_overrides"`
        FormularyAccess   map[string][]string         `yaml:"formulary_accessibility"`
}


type marketDrugOverride struct {
        DrugClass            string   `yaml:"drug_class"`
        SubstituteClass      string   `yaml:"substitute_class,omitempty"`
        InitiationMinEGFR    float64  `yaml:"initiation_min_egfr,omitempty"`
        ContinuationMinEGFR  float64  `yaml:"continuation_min_egfr,omitempty"`
        AvailabilityFlag     string   `yaml:"availability_flag,omitempty"`
        CostNote             string   `yaml:"cost_note,omitempty"`
}


func (f *RenalFormulary) GetRule(drugClass string) *models.RenalDrugRule {
        if r, ok := f.DrugRules[drugClass]; ok {
                return r
        }
        return nil
}


// LoadRenalFormulary loads shared rules and optionally merges market-specific overrides.
func LoadRenalFormulary(configDir string, market string) (*RenalFormulary, error) {
        // 1. Load shared base rules
        sharedPath := filepath.Join(configDir, "shared", "renal_dose_rules.yaml")
        data, err := os.ReadFile(sharedPath)
        if err != nil {
                return nil, fmt.Errorf("reading shared renal rules: %w", err)
        }


        var raw renalRulesYAML
        if err := yaml.Unmarshal(data, &raw); err != nil {
                return nil, fmt.Errorf("parsing shared renal rules: %w", err)
        }


        formulary := &RenalFormulary{
                DrugRules:     make(map[string]*models.RenalDrugRule),
                StaleEGFR:     raw.StaleEGFR,
                Accessibility: make(map[string]string),
        }
        for i := range raw.DrugRules {
                r := raw.DrugRules[i]
                formulary.DrugRules[r.DrugClass] = &r
        }


        // 2. If market specified, apply overrides
        if market == "" {
                return formulary, nil
        }


        overridePath := filepath.Join(configDir, market, "renal_overrides.yaml")
        if _, err := os.Stat(overridePath); os.IsNotExist(err) {
                return formulary, nil // no overrides for this market
        }


        overrideData, err := os.ReadFile(overridePath)
        if err != nil {
                return nil, fmt.Errorf("reading market renal overrides: %w", err)
        }


        var overrides marketRenalOverrideYAML
        if err := yaml.Unmarshal(overrideData, &overrides); err != nil {
                return nil, fmt.Errorf("parsing market renal overrides: %w", err)
        }


        // Merge stale eGFR config
        if overrides.StaleEGFR != nil {
                if overrides.StaleEGFR.WarningDays > 0 {
                        formulary.StaleEGFR.WarningDays = overrides.StaleEGFR.WarningDays
                }
                if overrides.StaleEGFR.CriticalDays > 0 {
                        formulary.StaleEGFR.CriticalDays = overrides.StaleEGFR.CriticalDays
                }
                formulary.StaleEGFR.HardBlockOnCritical = overrides.StaleEGFR.HardBlockOnCritical
        }


        // Merge drug rule overrides
        for _, ov := range overrides.DrugRuleOverrides {
                if rule, ok := formulary.DrugRules[ov.DrugClass]; ok {
                        if ov.SubstituteClass != "" {
                                rule.SubstituteClass = ov.SubstituteClass
                        }
                        if ov.InitiationMinEGFR > 0 {
                                rule.InitiationMinEGFR = ov.InitiationMinEGFR
                        }
                        if ov.ContinuationMinEGFR > 0 {
                                rule.ContinuationMinEGFR = ov.ContinuationMinEGFR
                        }
                }
        }


        // Build accessibility map
        for level, drugs := range overrides.FormularyAccess {
                for _, drug := range drugs {
                        formulary.Accessibility[drug] = level
                }
        }


        return formulary, nil
}

// kb-23-decision-cards/internal/services/renal_dose_gate.go
package services


import (
        "fmt"
        "strings"
        "time"


        "kb-decision-cards/internal/models"
)


// ActiveMedication represents a currently prescribed medication for gating evaluation.
type ActiveMedication struct {
        DrugClass     string  `json:"drug_class"`
        DrugName      string  `json:"drug_name,omitempty"`
        CurrentDoseMg float64 `json:"current_dose_mg,omitempty"`
        IsNew         bool    `json:"is_new,omitempty"` // true if this is a new prescription, not continuation
}


// RenalDoseGate is the core safety engine.
type RenalDoseGate struct {
        formulary *RenalFormulary
}


func NewRenalDoseGate(formulary *RenalFormulary) *RenalDoseGate {
        return &RenalDoseGate{formulary: formulary}
}


// Evaluate produces a gating verdict for a single medication against the patient's renal status.
func (g *RenalDoseGate) Evaluate(med ActiveMedication, renal models.RenalStatus) models.MedicationGatingResult {
        now := time.Now()


        result := models.MedicationGatingResult{
                DrugClass:     med.DrugClass,
                DrugName:      med.DrugName,
                CurrentDoseMg: med.CurrentDoseMg,
                EGFR:          renal.EGFR,
                EvaluatedAt:   now,
        }


        rule := g.formulary.GetRule(med.DrugClass)
        if rule == nil {
                result.Verdict = models.VerdictCleared
                result.Reason = "no renal dose rule defined for " + med.DrugClass
                return result
        }


        result.SourceGuideline = rule.SourceGuideline


        // 0. Stale eGFR check — if critical staleness and hard-block enabled
        daysSinceMeasurement := int(now.Sub(renal.EGFRMeasuredAt).Hours() / 24)
        if daysSinceMeasurement > g.formulary.StaleEGFR.CriticalDays && g.formulary.StaleEGFR.HardBlockOnCritical {
                result.Verdict = models.VerdictInsufficientData
                result.Reason = fmt.Sprintf("eGFR is stale (%d days since measurement, critical threshold %d days)", daysSinceMeasurement, g.formulary.StaleEGFR.CriticalDays)
                result.ClinicalAction = "Obtain current eGFR before continuing " + med.DrugClass
                result.MonitoringRequired = []string{"eGFR", "creatinine"}
                result.MonitoringFrequency = "IMMEDIATE"
                return result
        }
        // Soft stale warning (applies even without hard block)
        if daysSinceMeasurement > g.formulary.StaleEGFR.CriticalDays {
                result.Verdict = models.VerdictInsufficientData
                result.Reason = fmt.Sprintf("eGFR is stale (%d days since measurement)", daysSinceMeasurement)
                result.ClinicalAction = "Schedule eGFR measurement — last result may not reflect current renal function"
                result.MonitoringRequired = []string{"eGFR", "creatinine"}
                result.MonitoringFrequency = "URGENT"
                return result
        }


        // 1. Potassium co-gating (takes precedence for ACEi/ARB/MRA/FINERENONE)
        if rule.RequiresPotassiumCheck {
                if renal.Potassium == nil && renal.EGFR < rule.MonitorEscalateBelow {
                        // Missing potassium data with eGFR in danger zone
                        result.Verdict = models.VerdictMonitorEscalate
                        result.Reason = fmt.Sprintf("%s requires potassium monitoring at eGFR %.1f — potassium data unavailable", med.DrugClass, renal.EGFR)
                        result.ClinicalAction = "Check serum potassium before continuing " + med.DrugClass
                        result.MonitoringRequired = []string{"potassium", "eGFR", "creatinine"}
                        result.MonitoringFrequency = "WEEKLY"
                        return result
                }
                if renal.Potassium != nil && rule.PotassiumContraAbove > 0 && *renal.Potassium >= rule.PotassiumContraAbove {
                        result.Verdict = models.VerdictContraindicated
                        result.Reason = fmt.Sprintf("potassium %.1f ≥ %.1f threshold for %s", *renal.Potassium, rule.PotassiumContraAbove, med.DrugClass)
                        result.ClinicalAction = fmt.Sprintf("Hold %s until potassium < %.1f; recheck in 1 week", med.DrugClass, rule.PotassiumContraAbove)
                        result.MonitoringRequired = []string{"potassium", "eGFR"}
                        result.MonitoringFrequency = "WEEKLY"
                        return result
                }
        }


        // 2. Hard contraindication by eGFR
        if rule.ContraindicatedBelow > 0 && renal.EGFR < rule.ContraindicatedBelow {
                result.Verdict = models.VerdictContraindicated
                result.Reason = fmt.Sprintf("eGFR %.1f below %.1f contraindication threshold for %s", renal.EGFR, rule.ContraindicatedBelow, med.DrugClass)
                action := fmt.Sprintf("Discontinue %s", med.DrugClass)
                if rule.SubstituteClass != "" {
                        action += fmt.Sprintf("; consider switching to %s", rule.SubstituteClass)
                        result.SubstituteClass = rule.SubstituteClass
                }
                result.ClinicalAction = action
                return result
        }


        // 3. Efficacy cliff (e.g., thiazide at eGFR 30)
        if rule.EfficacyCliffBelow > 0 && renal.EGFR < rule.EfficacyCliffBelow {
                result.Verdict = models.VerdictDoseReduce // treated as dose-reduce to trigger switch
                result.Reason = fmt.Sprintf("eGFR %.1f below efficacy cliff %.1f for %s — drug has minimal effect", renal.EGFR, rule.EfficacyCliffBelow, med.DrugClass)
                action := fmt.Sprintf("Switch from %s", med.DrugClass)
                if rule.SubstituteClass != "" {
                        action += fmt.Sprintf(" to %s", rule.SubstituteClass)
                        result.SubstituteClass = rule.SubstituteClass
                }
                result.ClinicalAction = action
                return result
        }


        // 4. Dose reduction required
        if rule.DoseReduceBelow > 0 && renal.EGFR < rule.DoseReduceBelow {
                result.Verdict = models.VerdictDoseReduce
                result.Reason = fmt.Sprintf("eGFR %.1f below %.1f — dose reduction required for %s", renal.EGFR, rule.DoseReduceBelow, med.DrugClass)
                if rule.MaxDoseReducedMg > 0 {
                        maxDose := rule.MaxDoseReducedMg
                        result.MaxSafeDoseMg = &maxDose
                        result.ClinicalAction = fmt.Sprintf("Reduce %s to maximum %.0f mg/day", med.DrugClass, maxDose)
                } else {
                        result.ClinicalAction = fmt.Sprintf("Reduce %s dose — consult renal dosing guidelines", med.DrugClass)
                }
                return result
        }


        // 5. Monitor escalation (eGFR in caution zone)
        if rule.MonitorEscalateBelow > 0 && renal.EGFR < rule.MonitorEscalateBelow {
                result.Verdict = models.VerdictMonitorEscalate
                result.Reason = fmt.Sprintf("eGFR %.1f below %.1f — increased monitoring needed for %s", renal.EGFR, rule.MonitorEscalateBelow, med.DrugClass)
                result.ClinicalAction = "Increase renal monitoring frequency"
                result.MonitoringRequired = buildMonitoringList(rule)
                result.MonitoringFrequency = "QUARTERLY"
                if renal.EGFR < 45 {
                        result.MonitoringFrequency = "MONTHLY"
                }
                return result
        }


        // 6. All clear
        result.Verdict = models.VerdictCleared
        result.Reason = fmt.Sprintf("eGFR %.1f within safe range for %s", renal.EGFR, med.DrugClass)
        result.ClinicalAction = "No renal dose adjustment needed"
        return result
}


// EvaluatePatient produces a full gating report for all of a patient's medications.
func (g *RenalDoseGate) EvaluatePatient(patientID string, renal models.RenalStatus, meds []ActiveMedication) models.PatientGatingReport {
        report := models.PatientGatingReport{
                PatientID:   patientID,
                RenalStatus: renal,
        }


        // Check stale eGFR at patient level
        daysSince := int(time.Now().Sub(renal.EGFRMeasuredAt).Hours() / 24)
        if daysSince > g.formulary.StaleEGFR.WarningDays {
                report.StaleEGFR = true
                report.StaleEGFRDays = daysSince
        }


        for _, med := range meds {
                result := g.Evaluate(med, renal)
                report.MedicationResults = append(report.MedicationResults, result)


                switch result.Verdict {
                case models.VerdictContraindicated:
                        report.HasContraindicated = true
                case models.VerdictDoseReduce:
                        report.HasDoseReduce = true
                }
        }


        // Determine overall urgency
        report.OverallUrgency = g.determineOverallUrgency(report)


        return report
}


func (g *RenalDoseGate) determineOverallUrgency(report models.PatientGatingReport) string {
        if report.HasContraindicated {
                return "IMMEDIATE"
        }
        if report.HasDoseReduce {
                return "URGENT"

        for _, r := range report.MedicationResults {
                if r.Verdict == models.VerdictMonitorEscalate || r.Verdict == models.VerdictInsufficientData {
                        return "ROUTINE"
                }
        }
        return "SCHEDULED"
}

// BlockRecommendation checks if a proposed new medication should be blocked by renal status.
// This is the HARD GATE for the card builder — called BEFORE recommendation generation.
func (g *RenalDoseGate) BlockRecommendation(proposedDrugClass string, renal models.RenalStatus) (blocked bool, reason string, alternative string) {
        rule := g.formulary.GetRule(proposedDrugClass)
        if rule == nil {
                return false, "", ""
        }


        // Check initiation-specific threshold (e.g., Australia PBS SGLT2i ≥25 for new starts)
        if rule.InitiationMinEGFR > 0 && renal.EGFR < rule.InitiationMinEGFR {
                return true,
                        fmt.Sprintf("Cannot initiate %s — eGFR %.1f below initiation threshold %.1f (%s)",
                                proposedDrugClass, renal.EGFR, rule.InitiationMinEGFR, rule.SourceGuideline),
                        rule.SubstituteClass
        }


        // Check hard contraindication
        if rule.ContraindicatedBelow > 0 && renal.EGFR < rule.ContraindicatedBelow {
                return true,
                        fmt.Sprintf("Cannot prescribe %s — eGFR %.1f below contraindication threshold %.1f (%s)",
                                proposedDrugClass, renal.EGFR, rule.ContraindicatedBelow, rule.SourceGuideline),
                        rule.SubstituteClass
        }


        // Check potassium contraindication for new starts
        if rule.RequiresPotassiumCheck && rule.PotassiumContraAbove > 0 {
                if renal.Potassium != nil && *renal.Potassium >= rule.PotassiumContraAbove {
                        return true,
                                fmt.Sprintf("Cannot initiate %s — potassium %.1f ≥ %.1f", proposedDrugClass, *renal.Potassium, rule.PotassiumContraAbove),
                                ""
                }
        }


        // Check efficacy cliff for new starts
        if rule.EfficacyCliffBelow > 0 && renal.EGFR < rule.EfficacyCliffBelow {
                return true,
                        fmt.Sprintf("%s has minimal efficacy at eGFR %.1f (cliff at %.1f)", proposedDrugClass, renal.EGFR, rule.EfficacyCliffBelow),
                        rule.SubstituteClass
        }


        return false, "", ""
}

// kb-26-metabolic-digital-twin/internal/services/egfr_trajectory.go
package services


import (
        "math"
        "time"
)


type EGFRReading struct {
        Value     float64   `json:"value"`
        Timestamp time.Time `json:"timestamp"`
}


type EGFRTrajectory struct {
        SlopePerYear    float64 `json:"slope_per_year"`     // mL/min/1.73m²/year
        Classification  string  `json:"classification"`     // RAPID_DECLINE, MODERATE_DECLINE, STABLE, IMPROVING
        IsRapidDecliner bool    `json:"is_rapid_decliner"`  // slope < -5
        CurrentEGFR     float64 `json:"current_egfr"`
        DataPoints      int     `json:"data_points"`
        SpanDays        int     `json:"span_days"`
}


func ComputeEGFRTrajectory(readings []EGFRReading) EGFRTrajectory {
        if len(readings) < 2 {
                return EGFRTrajectory{Classification: "INSUFFICIENT_DATA", DataPoints: len(readings)}
        }


        sorted := make([]EGFRReading, len(readings))
        copy(sorted, readings)
        sortEGFRByTimestamp(sorted)


        baseTime := sorted[0].Timestamp
        n := float64(len(sorted))
        var sumX, sumY, sumXY, sumX2 float64


        for _, r := range sorted {
                x := r.Timestamp.Sub(baseTime).Hours() / (24.0 * 365.25) // years
                sumX += x
                sumY += r.Value
                sumXY += x * r.Value
                sumX2 += x * x
        }


        denom := n*sumX2 - sumX*sumX
        slope := 0.0
        if math.Abs(denom) > 1e-10 {
                slope = (n*sumXY - sumX*sumY) / denom
        }


        last := sorted[len(sorted)-1]
        spanDays := int(last.Timestamp.Sub(sorted[0].Timestamp).Hours() / 24)


        return EGFRTrajectory{
                SlopePerYear:    math.Round(slope*100) / 100,
                Classification:  classifyEGFRSlope(slope),
                IsRapidDecliner: slope < -5.0,
                CurrentEGFR:     last.Value,
                DataPoints:      len(sorted),
                SpanDays:        spanDays,
        }
}


func classifyEGFRSlope(slopePerYear float64) string {
        if slopePerYear < -5.0 {
                return "RAPID_DECLINE"
        }
        if slopePerYear < -1.0 {
                return "MODERATE_DECLINE"
        }
        if slopePerYear <= 1.0 {
                return "STABLE"
        }
        return "IMPROVING"
}


// ProjectTimeToThreshold returns months until eGFR crosses a threshold, given current slope.
// Returns (months, true) if crossing is projected with declining slope.
// Returns (0, true) if already below.
// Returns (0, false) if slope is non-negative (will not cross downward).
func ProjectTimeToThreshold(currentEGFR, slopePerYear, threshold float64) (float64, bool) {
        if currentEGFR <= threshold {
                return 0, true
        }
        if slopePerYear >= 0 {
                return 0, false // stable or improving — will not cross downward
        }
        yearsToThreshold := (currentEGFR - threshold) / math.Abs(slopePerYear)
        return yearsToThreshold * 12, true
}


func sortEGFRByTimestamp(pts []EGFRReading) {
        for i := 1; i < len(pts); i++ {
                for j := i; j > 0 && pts[j].Timestamp.Before(pts[j-1].Timestamp); j-- {
                        pts[j], pts[j-1] = pts[j-1], pts[j]
                }
        }
}

// AnticipatoryAlert warns that eGFR trajectory projects threshold crossing.
type AnticipatoryAlert struct {
        DrugClass      string  `json:"drug_class"`
        ThresholdType  string  `json:"threshold_type"`   // CONTRAINDICATION, DOSE_REDUCE, EFFICACY_CLIFF
        ThresholdEGFR  float64 `json:"threshold_egfr"`
        MonthsToReach  float64 `json:"months_to_reach"`
        Alert          string  `json:"alert"`
        RecommendedAction string `json:"recommended_action"`
}


// FindApproachingThresholds identifies medications whose renal thresholds
// the patient is projected to cross within the given horizon.
func FindApproachingThresholds(activeMedClasses []string, currentEGFR float64,
        slopePerYear float64, horizonMonths int, formulary *RenalFormulary) []AnticipatoryAlert {


        if slopePerYear >= 0 {
                return nil // not declining
        }


        var alerts []AnticipatoryAlert
        horizon := float64(horizonMonths)


        for _, drugClass := range activeMedClasses {
                rule := formulary.GetRule(drugClass)
                if rule == nil {
                        continue
                }


                // Check contraindication threshold
                if rule.ContraindicatedBelow > 0 && currentEGFR > rule.ContraindicatedBelow {
                        months, crosses := ProjectTimeToThreshold(currentEGFR, slopePerYear, rule.ContraindicatedBelow)
                        if crosses && months <= horizon && months > 0 {
                                alerts = append(alerts, AnticipatoryAlert{
                                        DrugClass:     drugClass,
                                        ThresholdType: "CONTRAINDICATION",
                                        ThresholdEGFR: rule.ContraindicatedBelow,
                                        MonthsToReach: months,
                                        Alert: fmt.Sprintf("%s approaching contraindication threshold (eGFR %.0f) in ~%.0f months at current trajectory",
                                                drugClass, rule.ContraindicatedBelow, months),
                                        RecommendedAction: fmt.Sprintf("Plan %s discontinuation/substitution; current eGFR %.1f declining at %.1f/year",
                                                drugClass, currentEGFR, slopePerYear),
                                })
                        }
                }


                // Check dose-reduce threshold
                if rule.DoseReduceBelow > 0 && currentEGFR > rule.DoseReduceBelow {
                        months, crosses := ProjectTimeToThreshold(currentEGFR, slopePerYear, rule.DoseReduceBelow)
                        if crosses && months <= horizon && months > 0 {
                                alerts = append(alerts, AnticipatoryAlert{
                                        DrugClass:     drugClass,
                                        ThresholdType: "DOSE_REDUCE",
                                        ThresholdEGFR: rule.DoseReduceBelow,
                                        MonthsToReach: months,
                                        Alert: fmt.Sprintf("%s approaching dose reduction threshold (eGFR %.0f) in ~%.0f months",
                                                drugClass, rule.DoseReduceBelow, months),
                                        RecommendedAction: fmt.Sprintf("Prepare dose reduction plan for %s", drugClass),
                                })
                        }
                }


                // Check efficacy cliff
                if rule.EfficacyCliffBelow > 0 && currentEGFR > rule.EfficacyCliffBelow {
                        months, crosses := ProjectTimeToThreshold(currentEGFR, slopePerYear, rule.EfficacyCliffBelow)
                        if crosses && months <= horizon && months > 0 {
                                alerts = append(alerts, AnticipatoryAlert{
                                        DrugClass:     drugClass,
                                        ThresholdType: "EFFICACY_CLIFF",
                                        ThresholdEGFR: rule.EfficacyCliffBelow,
                                        MonthsToReach: months,
                                        Alert: fmt.Sprintf("%s approaching efficacy cliff (eGFR %.0f) — plan switch to %s",
                                                drugClass, rule.EfficacyCliffBelow, rule.SubstituteClass),
                                        RecommendedAction: fmt.Sprintf("Switch from %s to %s before efficacy loss", drugClass, rule.SubstituteClass),
                                })
                        }
                }
        }


        return alerts
}


// ProjectTimeToThreshold is re-exported from KB-26 trajectory logic for use in KB-23.
func ProjectTimeToThreshold(currentEGFR, slopePerYear, threshold float64) (float64, bool) {
        if currentEGFR <= threshold {
                return 0, true
        }
        if slopePerYear >= 0 {
                return 0, false
        }
        years := (currentEGFR - threshold) / (-slopePerYear)
        return years * 12, true
}

package services


import (
        "fmt"
        "time"


        "kb-decision-cards/internal/models"
)


// StaleEGFRResult describes the staleness of a patient's eGFR data.
type StaleEGFRResult struct {
        IsStale          bool   `json:"is_stale"`
        IsCriticallyStale bool  `json:"is_critically_stale"`
        DaysSinceMeasure int    `json:"days_since_measurement"`
        RecommendedAction string `json:"recommended_action"`
        ExpectedFrequency string `json:"expected_frequency"`  // based on current eGFR
}


// DetectStaleEGFR evaluates whether a patient's eGFR data is current enough
// for safe clinical decision-making, adjusted for their CKD stage.
func DetectStaleEGFR(renal models.RenalStatus, formulary *RenalFormulary, activeMedClasses []string) StaleEGFRResult {
        daysSince := int(time.Now().Sub(renal.EGFRMeasuredAt).Hours() / 24)


        // Determine expected monitoring frequency based on eGFR level
        var expectedDays int
        var freqLabel string
        switch {
        case renal.EGFR < 30:
                expectedDays = 30
                freqLabel = "MONTHLY"
        case renal.EGFR < 45:
                expectedDays = 90
                freqLabel = "QUARTERLY"
        case renal.EGFR < 60:
                expectedDays = 180
                freqLabel = "BIANNUALLY"
        default:
                expectedDays = 365
                freqLabel = "ANNUALLY"
        }


        // Check if patient is on any renal-sensitive medication — tighten monitoring
        hasRenalSensitiveMed := false
        for _, mc := range activeMedClasses {
                if rule := formulary.GetRule(mc); rule != nil {
                        if rule.ContraindicatedBelow > 0 || rule.DoseReduceBelow > 0 {
                                hasRenalSensitiveMed = true
                                break
                        }
                }
        }
        if hasRenalSensitiveMed && expectedDays > 90 {
                expectedDays = 90 // at minimum quarterly for patients on renal-sensitive drugs
                freqLabel = "QUARTERLY (renal-sensitive medication)"
        }


        result := StaleEGFRResult{
                DaysSinceMeasure: daysSince,
                ExpectedFrequency: freqLabel,
        }


        if daysSince > formulary.StaleEGFR.CriticalDays {
                result.IsStale = true
                result.IsCriticallyStale = true
                result.RecommendedAction = fmt.Sprintf("URGENT: eGFR last measured %d days ago (critical threshold %d days). Schedule immediate renal function test.",
                        daysSince, formulary.StaleEGFR.CriticalDays)
        } else if daysSince > expectedDays {
                result.IsStale = true
                result.RecommendedAction = fmt.Sprintf("eGFR last measured %d days ago. Expected frequency: %s. Schedule renal function test.",
                        daysSince, freqLabel)
        }


        return result
}

// DetectAllConflicts runs both drug-domain conflict detection AND renal dose gating.
func DetectAllConflicts(activeMedClasses []string, renal *models.RenalStatus,
        gate *RenalDoseGate, formulary *RenalFormulary) EnrichedConflictReport {


        report := EnrichedConflictReport{
                DrugDomainConflicts: DetectDrugDomainConflicts(activeMedClasses),
        }


        if renal == nil || gate == nil {
                return report
        }


        // Build ActiveMedication list from classes
        meds := make([]ActiveMedication, len(activeMedClasses))
        for i, mc := range activeMedClasses {
                meds[i] = ActiveMedication{DrugClass: mc}
        }


        // Run renal gating
        gatingReport := gate.EvaluatePatient("", *renal, meds)
        report.RenalGatingReport = &gatingReport
        report.HasSafetyBlock = gatingReport.HasContraindicated


        // Run anticipatory alerts
        if renal.EGFRSlope < 0 {

                        activeMedClasses, renal.EGFR, renal.EGFRSlope, 12, formulary)
        }


        // Run stale eGFR check
        stale := DetectStaleEGFR(*renal, formulary, activeMedClasses)
        if stale.IsStale {
                report.StaleEGFR = &stale
        }


        return report
}

// kb-20-patient-profile/internal/api/renal_status_handlers.go
package api


import (
        "net/http"
        "time"


        "github.com/gin-gonic/gin"
)


// RenalStatusResponse is the renal data needed by KB-23 for gating.
type RenalStatusResponse struct {
        PatientID         string    `json:"patient_id"`
        EGFR              float64   `json:"egfr"`
        EGFRSlope         float64   `json:"egfr_slope"`          // mL/min/1.73m²/year
        EGFRMeasuredAt    time.Time `json:"egfr_measured_at"`
        EGFRDataPoints    int       `json:"egfr_data_points"`
        Potassium         *float64  `json:"potassium,omitempty"`
        PotassiumMeasuredAt *time.Time `json:"potassium_measured_at,omitempty"`
        ACR               *float64  `json:"acr,omitempty"`
        CKDStage          string    `json:"ckd_stage"`
        IsRapidDecliner   bool      `json:"is_rapid_decliner"`
        ActiveMedications []MedicationSummary `json:"active_medications"`
}


type MedicationSummary struct {
        DrugClass     string  `json:"drug_class"`
        DrugName      string  `json:"drug_name"`
        CurrentDoseMg float64 `json:"current_dose_mg"`
        StartDate     string  `json:"start_date"`
}


func (s *Server) getRenalStatus(c *gin.Context) {
        patientID := c.Param("id")


        var profile models.PatientProfile
        if err := s.db.DB.Where("patient_id = ? AND active = true", patientID).
                First(&profile).Error; err != nil {
                c.JSON(http.StatusNotFound, gin.H{"error": "patient not found"})
                return
        }


        // Fetch latest labs
        var latestEGFR struct {
                Value      float64
                RecordedAt time.Time
        }
        s.db.DB.Raw(`SELECT value, recorded_at FROM lab_entries 
                WHERE patient_id = ? AND lab_type = 'egfr' 
                ORDER BY recorded_at DESC LIMIT 1`, patientID).Scan(&latestEGFR)


        var latestPotassium struct {
                Value      float64
                RecordedAt time.Time
        }
        s.db.DB.Raw(`SELECT value, recorded_at FROM lab_entries 
                WHERE patient_id = ? AND lab_type = 'potassium' 
                ORDER BY recorded_at DESC LIMIT 1`, patientID).Scan(&latestPotassium)


        var latestACR struct{ Value float64 }
        s.db.DB.Raw(`SELECT value FROM lab_entries 
                WHERE patient_id = ? AND lab_type IN ('acr', 'uacr') 
                ORDER BY recorded_at DESC LIMIT 1`, patientID).Scan(&latestACR)


        // Count eGFR data points in trailing 12 months
        var egfrCount int64
        s.db.DB.Raw(`SELECT COUNT(*) FROM lab_entries 
                WHERE patient_id = ? AND lab_type = 'egfr' 
                AND recorded_at >= ?`, patientID, time.Now().Add(-365*24*time.Hour)).Scan(&egfrCount)


        // Compute CKD stage from eGFR
        ckdStage := classifyCKDStage(latestEGFR.Value)


        // Fetch active medications
        var meds []models.InterventionRecord
        s.db.DB.Where("patient_id = ? AND status = 'ACTIVE' AND intervention_type IN ('MEDICATION_START', 'DOSE_CHANGE')",
                patientID).Find(&meds)


        medSummaries := make([]MedicationSummary, 0, len(meds))
        for _, m := range meds {
                ms := MedicationSummary{StartDate: m.StartDate.Format("2006-01-02")}
                if m.DrugClass != nil { ms.DrugClass = *m.DrugClass }
                if m.DrugName != nil { ms.DrugName = *m.DrugName }
                if m.DoseChangeMg != nil { ms.CurrentDoseMg = *m.DoseChangeMg }
                medSummaries = append(medSummaries, ms)
        }


        resp := RenalStatusResponse{
                PatientID:         patientID,
                EGFR:              latestEGFR.Value,
                EGFRMeasuredAt:    latestEGFR.RecordedAt,
                EGFRDataPoints:    int(egfrCount),
                CKDStage:          ckdStage,
                ActiveMedications: medSummaries,
        }


        // Optional fields
        if latestPotassium.Value > 0 {
                resp.Potassium = &latestPotassium.Value
                resp.PotassiumMeasuredAt = &latestPotassium.RecordedAt
        }
        if latestACR.Value > 0 {
                resp.ACR = &latestACR.Value
        }


        // eGFR slope from profile (if computed by KB-26)
        if profile.EGFRSlope != nil {
                resp.EGFRSlope = *profile.EGFRSlope
                resp.IsRapidDecliner = *profile.EGFRSlope < -5.0
        }


        c.JSON(http.StatusOK, gin.H{"data": resp})
}


func classifyCKDStage(egfr float64) string {
        switch {
        case egfr >= 90: return "G1"
        case egfr >= 60: return "G2"
        case egfr >= 45: return "G3a"
        case egfr >= 30: return "G3b"
        case egfr >= 15: return "G4"
        default: return "G5"
        }
}

package services


import (
        "encoding/json"
        "fmt"
        "time"


        "go.uber.org/zap"
)


type RenalThresholdEvent struct {
        PatientID       string    `json:"patient_id"`
        EventType       string    `json:"event_type"` // RENAL_THRESHOLD_APPROACHING, RENAL_RAPID_DECLINE
        CurrentEGFR     float64   `json:"current_egfr"`
        SlopePerYear    float64   `json:"slope_per_year"`
        ThresholdEGFR   float64   `json:"threshold_egfr"`
        ProjectedMonths float64   `json:"projected_months"`
        DrugClasses     []string  `json:"drug_classes_affected"`
        Severity        string    `json:"severity"` // CRITICAL, WARNING, INFO
        Timestamp       time.Time `json:"timestamp"`
}


type RenalEventPublisher struct {
        eventBus EventBus
        logger   *zap.Logger
}


func NewRenalEventPublisher(eventBus EventBus, logger *zap.Logger) *RenalEventPublisher {
        return &RenalEventPublisher{eventBus: eventBus, logger: logger}
}


// EvaluateAndPublish checks eGFR trajectory against thresholds and publishes events.
func (p *RenalEventPublisher) EvaluateAndPublish(patientID string, trajectory EGFRTrajectory, activeMedClasses []string) error {
        if trajectory.Classification == "INSUFFICIENT_DATA" || trajectory.Classification == "STABLE" || trajectory.Classification == "IMPROVING" {
                return nil
        }


        // Check rapid decliner status
        if trajectory.IsRapidDecliner {
                event := RenalThresholdEvent{
                        PatientID:    patientID,
                        EventType:    "RENAL_RAPID_DECLINE",
                        CurrentEGFR:  trajectory.CurrentEGFR,
                        SlopePerYear: trajectory.SlopePerYear,
                        Severity:     "CRITICAL",
                        DrugClasses:  activeMedClasses,
                        Timestamp:    time.Now(),
                }
                if err := p.publish(event); err != nil {
                        return err
                }
        }


        // Check specific drug thresholds
        thresholds := map[string]float64{
                "METFORMIN":    30.0,
                "SGLT2i":       20.0,
                "SULFONYLUREA": 30.0,
                "MRA":          30.0,
                "EXENATIDE":    30.0,
                "FINERENONE":   25.0,
        }


        for _, drugClass := range activeMedClasses {
                threshold, exists := thresholds[drugClass]
                if !exists || trajectory.CurrentEGFR <= threshold {
                        continue
                }


                months, crosses := ProjectTimeToThreshold(trajectory.CurrentEGFR, trajectory.SlopePerYear, threshold)
                if crosses && months <= 12 && months > 0 {
                        severity := "WARNING"
                        if months <= 6 {
                                severity = "CRITICAL"
                        }


                        event := RenalThresholdEvent{
                                PatientID:       patientID,
                                EventType:       "RENAL_THRESHOLD_APPROACHING",
                                CurrentEGFR:     trajectory.CurrentEGFR,
                                SlopePerYear:    trajectory.SlopePerYear,
                                ThresholdEGFR:   threshold,
                                ProjectedMonths: months,
                                DrugClasses:     []string{drugClass},
                                Severity:        severity,
                                Timestamp:       time.Now(),
                        }
                        if err := p.publish(event); err != nil {
                                p.logger.Error("failed to publish renal threshold event", zap.Error(err))
                        }
                }
        }


        return nil
}


func (p *RenalEventPublisher) publish(event RenalThresholdEvent) error {
        data, err := json.Marshal(event)
        if err != nil {
                return fmt.Errorf("marshalling renal event: %w", err)
        }
        return p.eventBus.Publish("RENAL_THRESHOLD_EVENT", data)
}
